## EASProvisionDoc.DevicePasswordHistory property

İstemci tarafından yeniden kullanılmasını önlemek için saklanan önceden kullanılan minimum parola sayısını belirtir. Geçerli değerler aşağıda listelenmiştir: = 0 - Daha önce kullanılan parolaların depolanması gerekli değildir. &gt; 0 - Kullanılacak minimum önceden kullanılmış parola sayısı depolanmış. DevicePasswordHistory boş ise, istemci bu değeri 0 olarak değerlendirmelidir. DevicePasswordEnabled değeri TRUE olarak ayarlanırsa, istemci, parolanın süresi dolduktan sonra kullanıcının önceden depolanmış parolayı kullanmasına izin vermez. Değer DevicePasswordEnabled öğesi FALSE olarak ayarlanırsa, istemci bu özelliği YOKTURMALIDIR.
