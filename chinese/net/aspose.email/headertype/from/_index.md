## HeaderType.From property

此字段包含希望发送此消息的人的身份。 消息创建过程应默认此字段为单个经过身份验证的机器地址，指示组成消息的代理（人、系统或进程）。 如果不这样做，“发件人:”字段必须存在。如果“发件人:”字段默认采用这种方式，则“发件人:”字段是可选的，并且与“发件人:”字段是多余的。 示例: 来自:“某人先生”someone@somedomain.com

public static HeaderType From { get; }
