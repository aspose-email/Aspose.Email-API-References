## HeaderType.ReplyTo property

此标头字段用于指示发件人希望回复的位置。 不幸的是，这是模棱两可的，因为有不同类型的回复，发件人可能希望发送到不同的地址。 特别是，有个人回复只针对一个人，而群组回复针对阅读回复消息的整个群体（通常是邮件列表，由于语法不同，此处不能出现新闻组名称，请参阅“跟进”。）。

public static HeaderType ReplyTo { get; }
