## ActiveSyncTLClient.DeviceType property

设备类型由纯文本查询值的 device-type-spec ABNF 规则部分指定。 由设备类型 ABNF 规则表示的值是指定设备类型的任何字符串。 “SP”指定智能手机，“PPC”指定PocketPC。 其他客户端设备针对其特定设备类型发送唯一字符串。 来自客户端设备的每个请求必须包含相同的设备类型字符串。

public virtual string DeviceType { get; set; }
