## EASProvisionDoc.DevicePasswordExpiration property

指定密码过期前的最大天数。 DevicePasswordExpiration 可以为空，表示未设置密码过期策略。 有效值如下所示： = 0 - 密码不会过期。 &gt; 0 - 密码过期在指定的最大天数内。 如果 DevicePasswordExpiration 为空，客户端应将此值视为 0。 如果 DevicePasswordEnabled 的值设置为 FALSE，则客户端应忽略此属性。

public int? DevicePasswordExpiration { get; set; }
