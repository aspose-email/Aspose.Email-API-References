## RightsManagementLicense.ProgrammaticAccessAllowed property

指定第三方应用程序是否可以通过编程方式访问电子邮件的内容。 如果第三方应用程序可以以编程方式访问电子邮件的内容，则值为 TRUE；否则，错误。 TRUE 值表示受保护的内容是否可以被其他应用程序访问。 受保护的内容由邮件正文和附件组成。

public bool ProgrammaticAccessAllowed { get; set; }
