## IGraphClient.CopyNotebook method

将笔记本复制到目标文档库中的笔记本文件夹。如果文件夹不存在，则会创建该文件夹。 对于复制操作，您遵循异步调用模式:首先调用复制操作，然后轮询操作端点以获取结果。 权限 调用此 API 需要以下权限之一。 Delegated（工作或学校帐户）Notes.Create、Notes.ReadWrite、Notes.ReadWrite.All Delegated（个人 Microsoft 帐户）Notes.Create、Notes.ReadWrite Application Notes.ReadWrite.All

```csharp
public string CopyNotebook(string itemId, string groupId, string renameAs)
```

| 范围 | 类型 | 描述 |
| --- | --- | --- |
| itemId | String | 要复制的项目 ID |
| groupId | String | 要复制到的组的 id。仅在复制到 Office 365 组时使用。 |
| renameAs | String | 副本的名称。默认为现有项目的名称。 |

### 返回值

如果成功，此方法返回一个操作位置字符串。 您可以使用此值来获取操作的状态。
