## ItemOperationsFetchProperties.Range property

Spécifie la plage d'octets que le client peut recevoir en réponse à l'opération Fetch pour un élément de bibliothèque de documents. L'élément Range contient une valeur de chaîne au format mn, où m est inférieur à n, et m est la valeur minimale et n est la valeur maximale. La plage d'octets est indexée à zéro ; le premier octet est indiqué par un 0 (zéro). Si l'élément Range est omis dans la requête Fetch, l'élément entier est récupéré.

public string Range { get; set; }
