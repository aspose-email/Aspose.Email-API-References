## OOFMessage class

Spécifie le message OOF pour une audience particulière. Exchange 2007, Exchange 2010 et Exchange 2013 exigent que le message de réponse pour les audiences externes inconnues et connues soit le même. La propriété prend en charge les trois audiences suivantes pour un message OOF : Interne — Un utilisateur qui est dans la même organisation que l'utilisateur expéditeur. Connu externe — Un utilisateur qui est en dehors de l'organisation de l'utilisateur expéditeur, mais qui est représenté dans les contacts de l'utilisateur expéditeur. Inconnu externe — Un utilisateur qui est en dehors de l'utilisateur expéditeur organisation et n'est pas représenté dans les contacts de l'utilisateur expéditeur.
