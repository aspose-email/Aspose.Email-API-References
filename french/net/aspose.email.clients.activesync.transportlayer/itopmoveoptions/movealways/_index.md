## ItOpMoveOptions.MoveAlways property

Indique s'il faut toujours déplacer la conversation spécifiée, y compris tous les futurs e-mails de la conversation, vers le dossier spécifié par le DstFldId. Le MoveAlways DOIT être défini dans une requête ItemOperations lors de l'exécution d'une opération de déplacement sur une conversation. Une valeur d'élément Status de 155 est renvoyée si l'élément MoveAlways n'est pas défini dans la requête ItemOperations pour une opération de déplacement.

public bool? MoveAlways { get; set; }
