## BodyPart.Data property

Le contenu de l'élément Data est une chaîne au format spécifié par la propriété Type. Si la valeur du Type est RTF, la valeur de l'élément Data est encodée à l'aide de l'encodage base64. Si la propriété Truncated est définie sur true, les données de l'élément Data sont tronquées. La propriété EstimatedDataSize fournit une estimation approximative de la taille réelle du contenu complet de la chaîne de données.
