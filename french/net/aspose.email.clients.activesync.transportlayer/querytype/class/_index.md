## QueryType.Class property

Identifie la classe de l'élément. Les valeurs d'élément airsync:Class valides sont : - Tasks - Email - Calendar - Contacts - Notes - SMS Les classes suivantes sont prises en charge pour les recherches de boîte aux lettres lorsque la version du protocole est 12.1 : Email, Calendar, Contacts, Tâches. Les classes SMS et Notes ne sont disponibles que si la version du protocole est 14.0 ou 14.1. La requête Search peut inclure un ou plusieurs éléments Class dans la requête pour limiter le type de données incluses dans la réponse Search. Si un ou plusieurs éléments Class ne sont pas inclus dans la requête de recherche, le serveur renverra toutes les classes prises en charge. Si Class est inclus en tant qu'enfant d'un élément autre que l'élément And, le serveur répond avec une valeur Status de 8 (RechercheTropComplexe).
