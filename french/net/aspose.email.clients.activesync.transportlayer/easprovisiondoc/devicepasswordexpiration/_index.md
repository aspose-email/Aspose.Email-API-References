## EASProvisionDoc.DevicePasswordExpiration property

Spécifie le nombre maximal de jours avant l'expiration d'un mot de passe. Le DevicePasswordExpiration peut être nul, indiquant qu'aucune politique d'expiration de mot de passe n'est définie. Les valeurs valides sont répertoriées ci-dessous : = 0 - Les mots de passe n'expirent pas. &gt; 0 - Les mots de passe expirent dans le nombre maximal de jours spécifié. Si DevicePasswordExpiration est nul, un client DEVRAIT traiter cette valeur comme 0. Si la valeur de DevicePasswordEnabled est définie sur FALSE, le client DEVRAIT ignorer cette propriété.
