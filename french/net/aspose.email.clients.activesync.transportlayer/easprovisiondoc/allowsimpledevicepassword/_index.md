## EASProvisionDoc.AllowSimpleDevicePassword property

Spécifie si le client autorise les mots de passe simples. Un mot de passe simple est un mot de passe composé uniquement de caractères répétés ("2222") ou séquentiels ("abcd"). Si AllowSimpleDevicePassword est nul, un client DEVRAIT traiter cette valeur comme TRUE. Si la valeur de DevicePasswordEnabled est définie sur FALSE , le client DEVRAIT ignorer cette propriété.
