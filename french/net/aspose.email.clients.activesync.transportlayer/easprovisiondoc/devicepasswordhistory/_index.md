## EASProvisionDoc.DevicePasswordHistory property

Spécifie le nombre minimum de mots de passe précédemment utilisés stockés pour empêcher leur réutilisation par le client. Les valeurs valides sont répertoriées ci-dessous : = 0 - Le stockage des mots de passe précédemment utilisés n'est pas requis. &gt; 0 - Le nombre minimum de mots de passe précédemment utilisés à storage. Si DevicePasswordHistory est nul, alors un client DEVRAIT traiter cette valeur comme 0. Si la valeur de DevicePasswordEnabled est définie sur TRUE, le client interdit à l'utilisateur d'utiliser un mot de passe antérieur stocké après l'expiration d'un mot de passe. Si la valeur du DevicePasswordEnabled est défini sur FALSE, le client DEVRAIT ignorer cette propriété.
