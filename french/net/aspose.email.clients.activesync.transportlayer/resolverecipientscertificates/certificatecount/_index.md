## ResolveRecipientsCertificates.CertificateCount property

Spécifie le nombre de certificats valides trouvés pour le destinataire. Si une valeur de statut de 8 est renvoyée avec les certificats, CertificateCount spécifie le nombre de certificats de destinataire qui n'ont pas été renvoyés.
