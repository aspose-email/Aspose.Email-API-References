## SmartRequest.ReplaceMime property

Spécifie si le client envoie le message entier. Lorsque ReplaceMime est VRAI, le serveur NE DOIT PAS inclure le corps ou les pièces jointes du message d'origine transmis. Lorsque ReplaceMime est FALSE, le client DOIT ajouter le corps du message d'origine en pièces jointes au message sortant. Le client peut utiliser cette propriété pour indiquer si le message a été modifié en ligne ou si le message avait un texte de réponse/transfert ajouté au message source. Si le ReplaceMime est VRAI, le message a été modifié.
