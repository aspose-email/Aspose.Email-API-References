## SmartRequest.Mime property

Contient le message encodé MIME. Le contenu Mime est transféré sous la forme d'un BLOB opaque dans les balises WBXML, comme spécifié dans [WBXML1.2]. Si le message contient une demande de réunion, l'élément Mime contient les détails de la réunion dans Format iCalendar [MS-OXCICAL] ou format TNEF (Transport Neutral Encapsulation Format) [MS-OXTNEF]. Comme spécifié dans la section 3.4 de la [RFC2447], les demandes de réunion iCalendar ont un type de contenu de "texte/calendrier" avec le paramètre de méthode défini sur "REQUEST".
