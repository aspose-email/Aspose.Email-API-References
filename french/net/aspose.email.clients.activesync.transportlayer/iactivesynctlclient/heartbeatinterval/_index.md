## IActiveSyncTLClient.HeartbeatInterval property

L'élément HeartbeatInterval est un élément enfant de l'élément Ping dans les demandes et réponses de commande Ping. Dans les demandes de commande Ping, il spécifie la durée, en secondes, pendant laquelle le serveur DEVRAIT attendre avant d'envoyer une réponse si aucun nouvel élément n'est ajouté à l'ensemble de dossiers spécifié, comme spécifié dans la section 3.1.5.6. L'élément HeartbeatInterval est également renvoyé par le serveur avec un code d'état de 5 et spécifie l'intervalle minimum ou maximum autorisé lorsque le client a demandé un intervalle de pulsation en dehors de la plage acceptable.

public int HeartbeatInterval { get; set; }
