## SearchResult.LongId property

Spécifie un identifiant unique attribué par le serveur à chaque ensemble de résultats renvoyé. La valeur de LongId peut être utilisée comme ID long spécifié dans la demande de commande ItemOperations, la demande de commande SmartReply, la demande de commande SmartForward ou la demande de commande MeetingResponse pour référencer l'ensemble de résultats. Le client DOIT stocker la valeur de LongId sous la forme d'une chaîne opaque de 256 caractères maximum.

public string LongId { get; set; }
