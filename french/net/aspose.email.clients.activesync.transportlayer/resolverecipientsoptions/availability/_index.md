## ResolveRecipientsOptions.Availability property

Indique au serveur que des données de disponibilité sont demandées par le client et identifie l'heure de début et l'heure de fin des données de disponibilité à récupérer. Lorsque la disponibilité est incluse dans une requête ResolveRecipients, le serveur récupère la disponibilité informations pour les utilisateurs identifiés dans les éléments À inclus dans la demande, et renvoie les informations de disponibilité dans MergedFreeBusy dans la réponse. Si l'élément Disponibilité est inclus dans la demande ResolveRecipients, la demande DOIT également inclure une valeur StartTime et une valeur EndTime valides. Lorsque le serveur analyse la demande, le serveur résout d'abord les destinataires identifiés par les éléments À, puis détermine les informations de disponibilité des utilisateurs pour la période spécifiée, avant de renvoyer les données de disponibilité dans MergedFreeBusy.
