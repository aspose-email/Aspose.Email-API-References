## ResolveRecipientsOptions.MaxAmbiguousRecipients property

Limite le nombre de suggestions renvoyées pour chaque nœud destinataire ambigu dans la réponse. La valeur de MaxAmbiguousRecipients est limitée à une plage de 0 à 9999. Chaque nœud destinataire ambigu ne reçoit que ce nombre de suggestions et pas plus. Le nombre de destinataires, renvoyé dans RecipientCount, peut être utilisé par le client pour déterminer le nombre total de suggestions disponibles pour ce destinataire.

public int? MaxAmbiguousRecipients { get; set; }
