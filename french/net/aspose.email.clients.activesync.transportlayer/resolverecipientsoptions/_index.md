| [Availability](../../aspose.email.clients.activesync.transportlayer/resolverecipientsoptions/availability) { get; set; } | Indique au serveur que des données de disponibilité sont demandées par le client et identifie l'heure de début et l'heure de fin des données de disponibilité à récupérer. Lorsque la disponibilité est incluse dans une requête ResolveRecipients, le serveur récupère la disponibilité informations pour les utilisateurs identifiés dans les éléments À inclus dans la demande, et renvoie les informations de disponibilité dans MergedFreeBusy dans la réponse. Si l'élément Disponibilité est inclus dans la demande ResolveRecipients, la demande DOIT également inclure une valeur StartTime et une valeur EndTime valides. Lorsque le serveur analyse la demande, le serveur résout d'abord les destinataires identifiés par les éléments À, puis détermine les informations de disponibilité des utilisateurs pour la période spécifiée, avant de renvoyer les données de disponibilité dans MergedFreeBusy. |
| [CertificateRetrieval](../../aspose.email.clients.activesync.transportlayer/resolverecipientsoptions/certificateretrieval) { get; set; } | Spécifie si les certificats S/MIME DEVRAIENT être renvoyés par le serveur pour chaque destinataire résolu. |
| [MaxAmbiguousRecipients](../../aspose.email.clients.activesync.transportlayer/resolverecipientsoptions/maxambiguousrecipients) { get; set; } | Limite le nombre de suggestions renvoyées pour chaque nœud destinataire ambigu dans la réponse. La valeur de MaxAmbiguousRecipients est limitée à une plage de 0 à 9999. Chaque nœud destinataire ambigu ne reçoit que ce nombre de suggestions et pas plus. Le nombre de destinataires, renvoyé dans RecipientCount, peut être utilisé par le client pour déterminer le nombre total de suggestions disponibles pour ce destinataire. |
| [MaxCertificates](../../aspose.email.clients.activesync.transportlayer/resolverecipientsoptions/maxcertificates) { get; set; } | Limite le nombre total de certificats renvoyés par le serveur. La valeur de MaxCertificates est limitée à une plage de 0 à 9999. Cette limite garantit qu'aucun destinataire individuel ne reçoit un ensemble incomplet de certificats. Si la limite MaxCertificates est atteinte lors de l'énumération des certificats pour une liste d'adresses, cette liste d'adresses ne récupérera aucun certificat et une valeur d'état de 8 est renvoyée. |
| [Picture](../../aspose.email.clients.activesync.transportlayer/resolverecipientsoptions/picture) { get; set; } | Indique que le client demande que les photos de contact soient renvoyées dans la réponse du serveur.
