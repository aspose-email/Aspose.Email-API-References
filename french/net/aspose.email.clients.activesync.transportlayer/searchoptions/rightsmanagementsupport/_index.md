## SearchOptions.RightsManagementSupport property

Spécifie comment le serveur renvoie les e-mails gérés par des droits au client. Si la valeur est TRUE, le serveur décompressera et déchiffrera les e-mails gérés par des droits avant de les envoyer au client. Si la valeur est FALSE, le serveur ne décompressera ni ne déchiffrera les e-mails gérés par des droits avant de les envoyer au client. Si l'élément RightsManagementSupport n'est pas inclus dans un message de demande, une valeur par défaut de FALSE est supposée.

public bool? RightsManagementSupport { get; set; }
