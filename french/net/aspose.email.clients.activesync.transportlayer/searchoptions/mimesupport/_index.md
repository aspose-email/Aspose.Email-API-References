## SearchOptions.MIMESupport property

Active la prise en charge MIME pour les éléments de courrier électronique envoyés du serveur au client. Si l'élément airsync:MIMESupport est défini sur 'SendForSecureMIMEonly' (1) ou 'SendForAll' (2) dans la requête de recherche : - La propriété de le airsyncbase:BodyPreference, le Type, DEVRAIT être inclus dans la requête de recherche, contenant une valeur de 'MIME' (4) pour informer le serveur que l'appareil peut lire le BLOB MIME. - La réponse du serveur DOIT inclure le airsyncbase:Body, qui est un enfant de Properties. Le airsyncbase:Body DOIT contenir les propriétés suivantes dans une réponse de recherche S/MIME : - Le airsyncbase:Type avec une valeur de 'MIME' (4) pour informer l'appareil que les données sont un BLOB MIME. - Le airsyncbase :EstimatedDataSize pour spécifier la taille totale approximative des données. - Airsyncbase :Truncated pour indiquer si le BLOB MIME est tronqué. - Airsyncbase :Données contenant le BLOB MIME complet.
