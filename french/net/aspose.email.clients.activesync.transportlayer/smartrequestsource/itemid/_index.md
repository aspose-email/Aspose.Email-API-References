## SmartRequestSource.ItemId property

Spécifie l'ID d'élément pour le message source, qui est renvoyé dans la commande Sync. Si l'ItemId est présent, le FolderId DOIT également être présent si le message transféré ou répondu est stocké dans un dossier autre que le dossier Boîte de réception. La valeur de l'élément ItemId peut comporter jusqu'à 64 caractères.
