## RightsManagementLicense.Owner property

Spécifie si l'utilisateur est le propriétaire du message électronique. La valeur est TRUE si l'utilisateur est le propriétaire du message électronique ; sinon, FAUX. Une valeur TRUE indique que l'utilisateur authentifié a des droits de propriétaire sur ce message. Cet élément est utilisé uniquement à des fins de présentation d'informations. Les éléments autorisés (EditAllowed, ReplyAllowed, etc.) sont utilisés pour évaluer si une action particulière est autorisée ou restreinte.

public bool Owner { get; set; }
