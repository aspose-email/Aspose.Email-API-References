## ResolveRecipientsResponse.Recipients property

Représente les destinataires qui ont été résolus. Plusieurs destinataires sont renvoyés dans une réponse si le À spécifié dans la demande a été résolu en une liste de distribution ou trouvé ambigu. Le code d'état renvoyé dans la réponse peut être utilisé pour déterminer si le destinataire a été jugé ambigu. Le destinataire serait une correspondance suggérée si le destinataire spécifié dans la demande s'avérait être ambigu. Les certificats sont renvoyés si le client a demandé que des certificats soient renvoyés dans la réponse.
