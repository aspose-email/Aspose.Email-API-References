## SmtpClient.SucceededSending event

Ajoute ou supprime un abonné pour les notifications sur l'envoi réussi de messages électroniques. Veuillez noter que les notifications seront appelées à partir d'autres threads, vous devez donc implémenter un gestionnaire d'événements thread-safe. Notez également que cet événement est commun à l'instance du client de messagerie, dans le cas où le client fonctionne en mode multi-connexion, les messages de toutes les connexions seront traités via cet événement. Le client doit donc analyser dans son code les messages des événements pour comprendre quel message se rapporte à quelle opération.

```csharp
public event EventHandler<MailMessageEventArgs> SucceededSending;
```
