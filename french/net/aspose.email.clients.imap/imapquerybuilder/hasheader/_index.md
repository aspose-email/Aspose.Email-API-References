## ImapQueryBuilder.HasHeader method

Recherche les messages qui ont un en-tête avec le nom de champ spécifié et qui contient la chaîne spécifiée dans le texte de l'en-tête (ce qui vient après les deux-points). Si la chaîne à rechercher est de longueur nulle, cela correspond à tous les messages qui ont une ligne d'en-tête avec le nom de champ spécifié, quel que soit le contenu.
