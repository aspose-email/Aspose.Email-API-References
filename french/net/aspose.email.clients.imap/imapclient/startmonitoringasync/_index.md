## ImapClient.StartMonitoringAsync method

Démarre la surveillance des modifications de message pour le dossier spécifié.

```csharp
public Task StartMonitoringAsync(ImapMonitoringEventHandler callback, 
    ImapMonitoringErrorEventHandler errorCallback, string folderName = "Inbox")
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| callback | ImapMonitoringEventHandler | La fonction de rappel pour surveiller le fonctionnement. |
| errorCallback | ImapMonitoringErrorEventHandler | La fonction de rappel pour surveiller la gestion des erreurs. La surveillance du dossier spécifié est arrêtée lorsque ce rappel est appelé.
