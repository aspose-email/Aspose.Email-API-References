## ImapSpecialFolderTypes enumeration

Représente l'énumération des boîtes aux lettres à usage spécial Plus de détails voir dans RFC6154 http://tools.ietf.org/html/rfc6154

```csharp
public enum ImapSpecialFolderTypes
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| NotSpecified | `0` | Si l'extension IMAP LIST pour les boîtes aux lettres à usage spécial (RFC6154) est prise en charge, cela signifie que la boîte aux lettres est normale et n'est pas utilisée à des fins spéciales. |
| All | `1` | Cette boîte aux lettres présente tous les messages de la banque de messages de l'utilisateur. Les implémentations PEUVENT omettre certains messages, comme, peut-être, ceux de \Trash et \Junk. Lorsque cette utilisation spéciale est prise en charge, il est presque certain de représenter une boîte aux lettres virtuelle. |
| Archive | `2` | Cette boîte aux lettres est utilisée pour archiver les messages. La signification d'une boîte aux lettres "d'archivage" dépend du serveur ; généralement, il sera utilisé pour extraire les messages de la boîte de réception, ou les garder hors de portée de l'utilisateur, tout en les rendant accessibles. |
| Drafts | `3` | Cette boîte aux lettres est généralement utilisée pour contenir des brouillons de messages, des messages en cours de composition mais qui n'ont pas encore été envoyés. Dans certaines implémentations de serveur, il peut s'agir d'une boîte aux lettres virtuelle, contenant des messages d'autres boîtes aux lettres marqués du "\Draft" indicateur de messages. Alternativement, cela pourrait simplement être un conseil qu'un client place des brouillons ici. |
| Flagged | `4` | Cette boîte aux lettres présente tous les messages marqués d'une manière ou d'une autre comme "importants". Lorsque cette utilisation spéciale est prise en charge, il est probable qu'elle représente une boîte aux lettres virtuelle collectant les messages (provenant d'autres boîtes aux lettres) qui sont marqués avec l'indicateur de message "\Flagged". |
| Junk | `5` | Cette boîte aux lettres est l'endroit où les messages considérés comme du courrier indésirable sont conservés. Certaines implémentations de serveur peuvent placer des messages ici automatiquement. Alternativement, cela pourrait simplement être un conseil à un filtre anti-spam côté client. |
| Sent | `6` | Cette boîte aux lettres est utilisée pour conserver des copies des messages qui ont été envoyés. Certaines implémentations de serveur peuvent placer des messages ici automatiquement. Alternativement, cela pourrait simplement être un conseil qu'un client enregistre les messages envoyés ici. |
| Trash | `7` | Cette boîte aux lettres est utilisée pour conserver les messages qui ont été supprimés ou marqués pour suppression. Dans certaines implémentations de serveur, il peut s'agir d'une boîte aux lettres virtuelle contenant des messages d'autres boîtes aux lettres marqués de l'indicateur de message "\Deleted". Alternativement, cela pourrait simplement être un conseil qu'un client qui choisit de ne pas utiliser le modèle IMAP "\Deleted" devrait l'utiliser comme emplacement de la corbeille. Dans les implémentations de serveur qui attendent strictement le modèle IMAP "\Deleted", cette utilisation spéciale ne sera probablement pas prise en charge. |
| Important | `8` | Cette boîte aux lettres est utilisée pour conserver les messages qui ont été marqués comme importants. L'attribut de boîte aux lettres « \Important » indique que la boîte aux lettres contient des messages susceptibles d'être importants pour l'utilisateur. Dans une implémentation qui prend également en charge l'attribut « $Important ", cette utilisation spéciale est susceptible de représenter une boîte aux lettres virtuelle collectant les messages (provenant d'autres boîtes aux lettres) marqués du mot clé "$Important". Dans d'autres implémentations, le système peut y placer automatiquement des messages basés sur les mêmes types d'heuristiques que ceux notés pour le mot-clé "$Important". Les distinctions entre "\Important" et "\Flagged" pour les boîtes aux lettres sont similaires à celles entre "$Important" et "\Flagged" pour les messages. https://tools.ietf.org/html/rfc8457 |
