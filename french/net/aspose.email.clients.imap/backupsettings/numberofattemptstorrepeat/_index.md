## BackupSettings.NumberOfAttemptsToRrepeat property

Obtient ou définit la valeur qui définit le nombre de tentatives de répétition de l'opération ayant échoué Si une commande IMAP dans l'opération de sauvegarde renvoie un résultat d'échec, le client IMAP essaie de répéter cette opération à nouveau en fonction du nombre de fois défini. Par exemple, si l'opération FETCH renvoie une erreur AE_1_1_0243 FETCH 219 (BODY) AE_1_1_0243 NO[UNAVAILABLE] Le service FETCH n'est temporairement pas disponible Le client essaie de l'exécuter à nouveau.
