## MessageObject.GetIdForNamedProperty method

Obtient l'identifiant à utiliser pour la propriété nommée, les propriétés nommées sont des propriétés spéciales et doivent avoir leurs identifiants dans la plage [0x8000,0xfffe] alignés à partir de 0x8000 de manière séquentielle. Utilisez cette méthode pour trouver l'identifiant disponible car il pourrait être difficile de calculez-le vous-même.

```csharp
public int GetIdForNamedProperty()
```

### Return_Value

ID disponible pour la propriété nommée.
