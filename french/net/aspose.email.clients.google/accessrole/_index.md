Le rôle d'accès effectif que l'utilisateur authentifié a sur le calendrier.

```csharp
public enum AccessRole
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| none | `0` | Non défini |
| freeBusyReader | `1` | Fournit un accès en lecture aux informations de disponibilité. |
| reader | `2` | Fournit un accès en lecture au calendrier. Les événements privés apparaîtront aux utilisateurs disposant d'un accès lecteur, mais les détails de l'événement seront masqués. |
