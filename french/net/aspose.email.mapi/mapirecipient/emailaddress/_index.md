## MapiRecipient.EmailAddress property

Obtient ou définit l'adresse e-mail du destinataire ou de l'expéditeur du message .

```csharp
public string EmailAddress { get; set; }
```

### Valeur de la propriété

L'adresse e-mail.

### Exceptions

| exception | condition |
| --- | --- |
| ArgumentNullException | lève si l'adresse du destinataire est nulle ou vide. |
| ArgumentException | est lancé si l'adresse du destinataire n'est pas dans un format reconnu. |

### Remarques

Attention, cette propriété affiche une adresse au format SMTP. Cela signifie que si MapiRecipient.AddressType == "EX" et MapiPropertyTag.PR_EMAIL_ADDRESS contiennent une adresse dans le style Exchange, la valeur pour le remplissage de la propriété mapiRecipient.EmailAddress est extraite de PR_SMTP_ADDRESS. Lors de la définition d'une valeur, les valeurs de PR_SEARCH_KEY Les propriétés et PR_ENTRYID sont également mises à jour.
