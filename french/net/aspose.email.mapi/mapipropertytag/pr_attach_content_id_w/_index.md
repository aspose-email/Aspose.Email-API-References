## MapiPropertyTag.PR_ATTACH_CONTENT_ID_W field

Contient l'en-tête d'identification de contenu d'une pièce jointe de message MIME. Cette propriété est utilisée pour le support MHTML. Compilation UNICODE.

```csharp
public static readonly long PR_ATTACH_CONTENT_ID_W;
```

### Remarques

Il peut être utilisé pour distinguer les pièces jointes en ligne telles que les images de signature et les fichiers joints normaux. Si la pièce jointe a cette propriété, il s'agit d'une image intégrée.
