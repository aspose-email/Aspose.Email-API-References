## MailPrinter.CultureName property

Obtient ou définit le nom de la culture au format languagecode2-country/regioncode2.

```csharp
public string CultureName { get; set; }
```

### Valeur de la propriété

Une chaîne qui représente la culture actuelle.

### Remarques

languagecode2 est un code à deux lettres minuscules dérivé de la norme ISO 639-1. country/regioncode2 est dérivé de la norme ISO 3166 et se compose généralement de deux lettres majuscules.
