## CallInformation.SIPResponseCode property

Code de réponse SIP Cette liste détaille tous les codes de réponse SIP définis dans les RFC IETF et enregistrés auprès de l'IANA au 18 avril 2013. Elle comprend également les codes de réponse SIP définis dans les RFC SIP obsolètes (en particulier, RFC 2543), qui sont donc non enregistré auprès de l'IANA ; elles sont explicitement notées comme telles. 1xx—Réponses provisoires 2xx—Réponses réussies 3xx—Réponses de redirection 4xx—Réponses d'échec client 5xx—Réponses d'échec serveur 6xx—Réponses d'échec global
