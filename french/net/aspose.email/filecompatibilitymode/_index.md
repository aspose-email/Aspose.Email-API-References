## FileCompatibilityMode enumeration

Définit les conversions internes, qui sont obligatoirement à faire lors du chargement de ou de l'enregistrement d'un message. Par défaut, CR n'est pas remplacé par CRLF, La pièce jointe Tnef n'est pas enregistrée.

```csharp
[Flags]
public enum FileCompatibilityMode
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| None | `0` | Aucune valeur n'est définie. |
| AllowCROnly | `1` | Montre la nécessité du remplacement forcé des séparateurs de ligne par CRLF. |
| PreserveTnefAttachments | `2` | Contrôle le comportement de chargement/enregistrement des pièces jointes TNEF |
