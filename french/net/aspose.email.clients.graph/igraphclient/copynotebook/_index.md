## IGraphClient.CopyNotebook method

Copie un bloc-notes dans le dossier Blocs-notes de la bibliothèque Documents de destination. Le dossier est créé s'il n'existe pas. Pour les opérations de copie, vous suivez un modèle d'appel asynchrone : appelez d'abord l'action de copie, puis interrogez le point de terminaison de l'opération pour obtenir le résultat. Permissions L'une des autorisations suivantes est requise pour appeler cette API. Délégué (compte professionnel ou scolaire) Notes.Create, Notes.ReadWrite, Notes.ReadWrite.All Délégué (compte Microsoft personnel) Notes.Create, Notes.ReadWrite Application Notes.ReadWrite.All

```csharp
public string CopyNotebook(string itemId, string groupId, string renameAs)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| itemId | String | ID de l'article à copier |
| groupId | String | ID du groupe vers lequel copier. À utiliser uniquement lors de la copie vers un groupe Office 365. |
| renameAs | String | Le nom de la copie. Par défaut, le nom de l'élément existant. |

### Return_Value

En cas de succès, cette méthode renvoie une chaîne Operation-Location. Vous pouvez utiliser cette valeur pour obtenir le statut de l'opération.
