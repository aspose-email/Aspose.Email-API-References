## CreateOrUpdateOverride(MailAddress, ClassificationType) {#createorupdateoverride_1}

Создать переопределение для отправителя, идентифицируемого по SMTP-адресу. Будущие сообщения с этого SMTP-адреса будут последовательно классифицироваться в соответствии с указанными в переопределении. - Максимальное количество переопределений, поддерживаемых для почтового ящика, составляет 1000, исходя из уникальных SMTP-адресов отправителя.

```csharp
public ClassificationOverride CreateOrUpdateOverride(MailAddress sender, 
    ClassificationType classifyAs)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| sender | MailAddress | Информация об адресе электронной почты отправителя, для которого создается переопределение. |
| classifyAs | ClassificationType | Значение, указывающее, как всегда следует классифицировать входящие сообщения от определенного отправителя. |
