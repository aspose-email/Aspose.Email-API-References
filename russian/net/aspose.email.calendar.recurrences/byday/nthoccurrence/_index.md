## ByDay.NthOccurrence property

Получает или задает n-е вхождение дня недели.

```csharp
public int NthOccurrence { get; set; }
```

### Примечания

Допустимый диапазон для этого свойства:от -53 до 53.

Положительные значения представляют N-е появление с начала периода, например, NthOccurrence = 1, представляет 1-е появление дня недели.

Отрицательные значения представляют N-е вхождение с конца периода, например, NthOccurrence = -1, представляет последнее вхождение в день неделя.

Когда NthOccurrence равен нулю, он представляет все вхождения указанного дня недели. Например, BYDAY=MO имеет NthOccurrence ноль и представляет все понедельники в наборе.
