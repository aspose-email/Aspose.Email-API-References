## SearchOptions.RightsManagementSupport property

Указывает, как сервер возвращает клиенту сообщения электронной почты с управляемыми правами. Если значение равно TRUE, сервер распаковывает и расшифровывает сообщения электронной почты с управляемыми правами перед их отправкой клиенту. Если значение равно FALSE, сервер не будет распаковывать или расшифровывать сообщения электронной почты с управляемыми правами перед их отправкой клиенту. Если элемент RightsManagementSupport не включен в сообщение запроса, предполагается значение по умолчанию FALSE.

public bool? RightsManagementSupport { get; set; }
