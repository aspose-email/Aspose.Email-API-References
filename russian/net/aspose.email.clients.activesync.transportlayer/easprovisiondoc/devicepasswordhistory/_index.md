## EASProvisionDoc.DevicePasswordHistory property

Определяет минимальное количество ранее использовавшихся паролей, сохраняемых для предотвращения повторного использования клиентом. Допустимые значения перечислены ниже: Stored. Если DevicePasswordHistory имеет значение null, то клиент ДОЛЖЕН рассматривать это значение как 0. Если значение DevicePasswordEnabled установлено в TRUE, клиент запрещает пользователю использовать сохраненный предыдущий пароль после истечения срока действия пароля. Если значение для DevicePasswordEnabled установлено значение FALSE, клиент ДОЛЖЕН игнорировать это свойство.
