## EASProvisionDoc.DevicePasswordExpiration property

Указывает максимальное количество дней до истечения срока действия пароля. DevicePasswordExpiration может быть нулевым, указывая на то, что политика истечения срока действия пароля не установлена. Допустимые значения перечислены ниже: = 0 - Срок действия паролей не ограничен. &gt; 0 — Срок действия паролей истекает через указанное максимальное количество дней. Если DevicePasswordExpiration имеет значение null, клиент ДОЛЖЕН рассматривать это значение как 0. Если значение DevicePasswordEnabled установлено в FALSE, клиент ДОЛЖЕН игнорировать это свойство.

public int? DevicePasswordExpiration { get; set; }
