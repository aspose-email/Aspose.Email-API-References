## SyncRequest.Wait property

Указывает количество минут, на которое сервер ДОЛЖЕН задержать ответ, если во включенные папки не добавляются новые элементы. сервер отправит ответ, который включает в себя значение состояния 14 и предел. Либо ожидание, либо HeartbeatInterval могут быть включены в запрос команды синхронизации, но не оба. Если оба включены, ответ сервера будет содержать значение состояния 4.

public int? Wait { get; set; }
