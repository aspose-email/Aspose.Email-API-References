## SyncAddServerOperation.ClientId property

Содержит уникальный идентификатор, генерируемый клиентом для временной идентификации нового объекта, создаваемого с помощью элемента Add. Клиент включает элемент ClientId в запрос на добавление элемента, который он отправляет на сервер. Ответ сервера содержит элемент Add, содержащий исходный идентификатор клиента и новый идентификатор сервера, назначенный объекту, который заменяет идентификатор клиента в качестве постоянного идентификатора объекта. Элемент ClientId — это уникальный идентификатор, содержащий до 64 цифр и букв. Клиент генерирует этот идентификатор. Значение должно быть уникальным для устройства только во время выполнения запроса на синхронизацию, добавляющего объект на сервер. Клиент сохраняет идентификаторы клиентов до тех пор, пока сеанс синхронизации не будет успешно завершен, чтобы облегчить восстановление в случае сбоя процесса синхронизации.
