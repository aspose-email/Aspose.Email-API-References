| [ClientId](../../aspose.email.clients.activesync.transportlayer/syncaddserveroperation/clientid) { get; set; } | Содержит уникальный идентификатор, созданный клиентом для временной идентификации нового объекта, создаваемого с помощью элемента Add. Клиент включает элемент ClientId в запрос на добавление элемента, который он отправляет на сервер. Ответ сервера содержит элемент Add, который содержит исходный идентификатор клиента и новый идентификатор сервера, назначенный объекту, который заменяет идентификатор клиента в качестве постоянного идентификатора объекта. Элемент ClientId представляет собой уникальный идентификатор, состоящий из 64 цифры и буквы. Клиент генерирует этот идентификатор. Значение должно быть уникальным для устройства только во время выполнения запроса на синхронизацию, который добавляет объект на сервер. Клиент сохраняет идентификаторы клиентов до тех пор, пока сеанс синхронизации не будет успешно завершен, чтобы облегчить восстановление в случае сбоя процесса синхронизации. |
