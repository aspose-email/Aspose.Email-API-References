## ItemOperationsFetchProperties.Range property

Задает диапазон байтов, которые клиент может получить в ответ на операцию Fetch для элемента библиотеки документов. Элемент Range содержит строковое значение в формате mn, где m меньше n, m — минимальное значение, а n — максимальное значение. Диапазон байтов имеет нулевой индекс; первый байт обозначается 0 (нулем). Если элемент Range опущен в запросе на выборку, извлекается весь элемент.

```csharp
public string Range { get; set; }
```
