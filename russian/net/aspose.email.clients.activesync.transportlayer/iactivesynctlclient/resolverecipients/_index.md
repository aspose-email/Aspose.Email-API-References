## IActiveSyncTLClient.ResolveRecipients method

ResolveRecipients используется для разрешения списка предоставленных получателей, для получения их информации о занятости и, при необходимости, для получения их сертификатов S/MIME, чтобы клиенты могли отправлять зашифрованные сообщения электронной почты S/MIME. Получение информации о занятости использование элемента Availability в команде ResolveRecipients не поддерживается, если используется версия протокола 12.1.
