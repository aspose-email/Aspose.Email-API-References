## SmartRequestSource.ItemId property

Указывает идентификатор элемента для исходного сообщения, который возвращается командой Sync. Если ItemId присутствует, FolderId также ДОЛЖЕН присутствовать, если пересылаемое сообщение или ответ на него хранится в папке, отличной от папки "Входящие". Значение элемента ItemId может иметь длину до 64 символов.

public string ItemId { get; set; }
