## IBaseActiveSyncTLClient.DeviceType property

Тип устройства определяется частью правила ABNF спецификации типа устройства в значении простого текстового запроса. Значение, представленное правилом ABNF типа устройства, представляет собой любую строку, указывающую тип устройства. "SP" указывает SmartPhone, а "PPC" указывает PocketPC. Другие клиентские устройства отправляют уникальные строки для своего конкретного типа устройства. Каждый запрос от клиентского устройства ДОЛЖЕН включать одну и ту же строку типа устройства.

public string DeviceType { get; set; }
