## AccountInformation.AccountId property

Идентифицирует учетную запись. Основная учетная запись, определенная элементом PrimarySmtpAddress, не имеет значения AccountId. AccountId не поддерживается, если используется версия протокола 12.1 или 14.0.

public string AccountId { get; set; }
