## AccountInformation.SendDisabled property

Указывает, может ли клиент отправлять сообщения, используя данную учетную запись. Значение SendDisabled равно TRUE, если клиент не может отправить с использованием данной учетной записи; в противном случае ЛОЖЬ. SendDisabled не поддерживается, если используется версия протокола 12.1 или 14.0.

public bool? SendDisabled { get; set; }
