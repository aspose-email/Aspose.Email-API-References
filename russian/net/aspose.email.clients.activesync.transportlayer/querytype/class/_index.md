## QueryType.Class property

Идентифицирует класс предмета. Допустимые значения элемента airsync:Class: - Задачи - Электронная почта - Календарь - Контакты - Заметки - SMS Следующие классы поддерживаются для поиска в почтовых ящиках, когда версия протокола 12.1:Электронная почта, Календарь, Контакты, Задачи. Классы SMS и Notes доступны только при версии протокола 14.0 или 14.1. Запрос на поиск может включать в себя один или несколько элементов класса, чтобы ограничить тип данных, включенных в ответ на поиск. Если один или несколько элементов класса не включены в поисковый запрос, сервер вернет все поддерживаемые классы. Если класс включен в качестве дочернего элемента любого элемента, кроме элемента And, сервер отвечает значением состояния 8 (SearchTooComplex).
