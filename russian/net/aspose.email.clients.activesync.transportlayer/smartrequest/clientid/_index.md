## SmartRequest.ClientId property

Указывает уникальный идентификатор сообщения клиента (MID). Значение ClientId может иметь длину до 40 символов и ДОЛЖНО быть уникальным для каждого сообщения, , поскольку сервер будет использовать значение ClientId для идентификации повторяющихся сообщений. Значение ClientId может быть простым счетчиком, увеличивающимся для каждого нового сообщения.
