## ItOpMoveOptions.MoveAlways property

Указывает, следует ли всегда перемещать указанную беседу, включая все будущие сообщения электронной почты в беседе, в папку, указанную DstFldId. MoveAlways ДОЛЖЕН быть установлен в запросе ItemOperations при выполнении операции перемещения в диалоге. Значение элемента Status, равное 155, возвращается, если элемент MoveAlways не установлен в запросе ItemOperations для операции перемещения.

```csharp
public bool? MoveAlways { get; set; }
```
