## SyncCollectionRequest.ConversationMode property

Указывает, включать ли элементы, включенные в модальность диалога, в результаты ответа синхронизации. Установка значения элемента ConversationMode в TRUE приводит к получению всех сообщений электронной почты, соответствующих беседам, полученным в рамках указанного фильтра дат. Однако, несмотря на то, что тело электронных писем вне этого фильтра по времени не будет получено, предварительный просмотр текста будет получен, если предварительный просмотр был запрошен. Установка значения элемента ConversationMode значение FALSE в запросе синхронизации приводит к синхронизации элементов, соответствующих критериям значения элемента FilterType (раздел 2.2.3.64). Установка значения элемента ConversationMode в TRUE расширяет результирующий набор, чтобы также включить любые элементы с идентичными значениями email2:ConversationId ([MS-ASEMAIL], раздел 2.2.2.14) для тех, что указаны в набор результатов FilterType. Значение элемента ConversationMode не влияет на элементы за пределами коллекции, указанной элементом CollectionId (раздел 2.2.3.30.5); результирующий набор всегда ограничен элементами указанной коллекции. Значение элемента ConversationMode только ограничивает или расширяет результаты, определенные значением элемента FilterType.
