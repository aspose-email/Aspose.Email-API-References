## SyncCollectionRequest.WindowSize property

Указывает максимальное количество измененных элементов в коллекции или запросе, которые СЛЕДУЕТ включать в ответ синхронизации. Если WindowSize не определен, сервер ведет себя так, как если бы был отправлен элемент WindowSize со значением 100. Сервер интерпретирует значение 0 (ноль) и значения выше 512 как 512.

public int? WindowSize { get; set; }
