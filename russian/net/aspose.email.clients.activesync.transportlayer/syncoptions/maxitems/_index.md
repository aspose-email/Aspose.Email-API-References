## SyncOptions.MaxItems property

Задает максимальное количество получателей (т. е. первые N наиболее часто используемых получателей) для синхронизации из кэша информации о получателях. Этот элемент ДОЛЖЕН быть включен в запрос только тогда, когда значение элемента CollectionId (раздел 2.2.3.30.5) отображается в кэш информации получателя. Значение элемента MaxItems не указывает максимальную оценку добавлений и удалений, которые нужно сделать в кэше информации о получателях, оно указывает только количество получателей, которые необходимо синхронизировать. Полная замена каждого получателя удвоит количество MaxItems или элементов в магазине, поскольку каждое обновление получателя требует удаления и добавления.

```csharp
public int? MaxItems { get; set; }
```
