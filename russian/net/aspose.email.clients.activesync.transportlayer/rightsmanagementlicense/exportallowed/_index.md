## RightsManagementLicense.ExportAllowed property

Указывает, может ли пользователь удалить защиту IRM для сообщения электронной почты. Значение равно TRUE, если пользователь может удалить защиту IRM, когда пользователь пересылает, отвечает или отвечает на все сообщения электронной почты; в противном случае — FALSE. Если сообщение электронной почты с управлением правами пересылается или на него отвечает команда SmartForward или SmartReply, оцениваются следующие условия: для сообщения установлен шаблон «Нет ограничений» (значение TemplateID «00000000-0000-0000-0000-000000000000») Если выполняются оба условия, защита IRM снимается с исходящего сообщения. Исходное сообщение сохраняет защиту IRM.

public bool ExportAllowed { get; set; }
