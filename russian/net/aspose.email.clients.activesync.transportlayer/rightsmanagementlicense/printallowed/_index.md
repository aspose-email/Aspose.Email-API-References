## RightsManagementLicense.PrintAllowed property

Указывает, может ли пользователь распечатать сообщение электронной почты. Этот элемент не указывает на поддержку клиентом печати сообщения электронной почты; он только указывает, можно ли распечатать сообщение электронной почты, если клиент поддерживает печать. Значение TRUE, если электронное письмо может быть распечатано пользователем; в противном случае ЛОЖЬ.

public bool PrintAllowed { get; set; }
