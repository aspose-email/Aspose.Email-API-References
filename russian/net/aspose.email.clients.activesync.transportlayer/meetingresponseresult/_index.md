| [CalendarId](../../aspose.email.clients.activesync.transportlayer/meetingresponseresult/calendarid) { get; set; } | Указывает идентификатор сервера элемента календаря. Значение элемента CalendarId может иметь длину до 64 символов. Элемент CalendarId включается в ответ на команду MeetingResponse, который отправляется клиенту, если приглашение на собрание не было отклонено. Если собрание принято или предварительно принято, сервер добавляет новый элемент в календарь и возвращает идентификатор своего сервера в элементе CalendarId в ответе. Если клиент создал предварительный элемент календаря совещаний, клиент обновляет этот элемент с помощью нового идентификатора сервера. Клиент также меняет статус занятости с "предварительно" на "занят". Когда собрание принято, сервер также создает новый элемент календаря с тем же идентификатором сервера. Это означает, что существует конфликт, который будет разрешен при следующей синхронизации календаря. Если собрание отклонено, ответ не содержит элемент CalendarId. |
