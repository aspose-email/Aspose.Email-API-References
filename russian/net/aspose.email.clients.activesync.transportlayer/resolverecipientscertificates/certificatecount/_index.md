## ResolveRecipientsCertificates.CertificateCount property

Указывает количество действительных сертификатов, которые были найдены для получателя. Если значение состояния 8 возвращается с сертификатами, CertificateCount указывает количество сертификатов получателей, которые не были возвращены.
