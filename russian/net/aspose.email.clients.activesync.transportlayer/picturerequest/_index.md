## PictureRequest class

Указывает, что клиент запрашивает возврат фотографий в ответе сервера. Изображение не поддерживается при версии протокола 12.1 или 14.0. Содержит данные, связанные с запросом фотографий.

## Характеристики

| Имя | Описание |
| --- | --- |
| [MaxPictures](../../aspose.email.clients.activesync.transportlayer/picturerequest/maxpictures) { get; set; } | Ограничивает количество фотографий контактов, возвращаемых в ответе сервера. MaxPictures не поддерживается, если используется версия протокола 12.1 или 14.0. Сервер возвращает первые N результатов с фотографиями контактов, где N — значение MaxPictures. После достижения предела MaxPictures сервер возвращает значение состояния 173 (NoPicture), если у контакта нет фотографии, значение состояния 174 (PictureTooLarge), если фотография контакта слишком большая, или значение состояния 175 (PictureLimitReached). ), если у контакта есть фото, но достигнут лимит MaxPictures. Обратите внимание, что MaxPictures определяет количество фотографий контактов, возвращаемых по запросу. Таким образом, если клиент включает трех получателей для разрешения и задает для параметра MaxPictures значение 3 в командном запросе ResolveRecipients, может быть возвращено не более 9 фотографий контактов. |
| [MaxSize](../../aspose.email.clients.activesync.transportlayer/picturerequest/maxsize) { get; set; } | Указывает максимальный размер отдельной фотографии контакта, возвращаемой в ответе, в байтах. MaxSize не поддерживается, если используется версия протокола 12.1 или 14.0. Максимальное значение MaxSize составляет 100 КБ или 102400 байт. |
