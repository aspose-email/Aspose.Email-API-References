## PictureRequest.MaxPictures property

Ограничивает количество фотографий контактов, возвращаемых в ответе сервера. MaxPictures не поддерживается при версии протокола 12.1 или 14.0. Сервер возвращает первые N результатов, содержащих фотографии контактов, где N — значение MaxPictures. После достижения предела MaxPictures сервер возвращает значение состояния 173 (NoPicture), если у контакта нет фотографии, значение состояния 174 (PictureTooLarge), если фотография контакта слишком большая, или значение состояния 175 (PictureLimitReached), если у контакта есть photo, но достигнут предел MaxPictures. Обратите внимание, что MaxPictures определяет количество фотографий контактов, возвращаемых по запросу. Таким образом, если клиент включает трех получателей для разрешения и устанавливает значение MaxPictures равным 3 в командном запросе ResolveRecipients, может быть возвращено максимум 9 фотографий контактов.
