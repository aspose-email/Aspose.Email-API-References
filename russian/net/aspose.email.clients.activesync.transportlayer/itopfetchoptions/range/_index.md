## ItOpFetchOptions.Range property

Задает диапазон байтов, которые клиент может получить в ответ на операцию Fetch для элемента библиотеки документов. Элемент Range является необязательным полем. Элемент Range содержит строковое значение в формате MN, где M меньше N, а M — минимальное значение, а N — максимальное значение. Диапазон байтов имеет нулевой индекс; первый байт обозначается 0 (нулем). Если Range опущен в запросе на выборку, извлекается весь элемент. Если FileReference присутствует в запросе, то элемент Range является единственным допустимым дочерним элементом элемента Options.

public string Range { get; set; }
