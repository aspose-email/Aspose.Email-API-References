## ItOpFetchOptions.RightsManagementSupport property

Указывает, как сервер возвращает клиенту сообщения электронной почты с управляемыми правами. Если значение этого элемента равно TRUE, сервер будет распаковывать и расшифровывать сообщения электронной почты с управляемыми правами перед их отправкой клиенту, как указано в разделе 3.2.4.3. Если значение равно FALSE (0), сервер не будет распаковывать или расшифровывать сообщения электронной почты с управляемыми правами перед их отправкой клиенту. Если элемент RightsManagementSupport не определен, предполагается значение по умолчанию FALSE.

public bool? RightsManagementSupport { get; set; }
