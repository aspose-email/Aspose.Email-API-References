Получает или задает текст сообщения в формате RTF.

public string BodyRtf { get; set; }

### Стоимость имущества

Строка, представляющая тело сообщения rtf.

### Примечания

При установке значения обновляются значения свойств PR_RTF_COMPRESSED, PR_RTF_DECOMPRESSES, PR_BODY. Устанавливаемое строковое значение должно иметь формат RTF. Таким образом, если необходимо установить значение в формате HTML, значение должно быть предварительно закодировано в RTF, в соответствии со спецификацией RTF Extensions. Чтобы быстро установить содержимое основного сообщения в форматах HTML или Plain Text, используйте метод SetBodyContent. При установке нулевого значения или пустой строки значения свойств BodyRtf и Body устанавливаются нулевыми.
