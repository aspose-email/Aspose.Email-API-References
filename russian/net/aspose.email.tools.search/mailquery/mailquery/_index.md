### Примечания

Строка запроса должна иметь следующий вид.

Пример простого выражения:

'&lt;Имя поля&gt;' &lt;Оператор сравнения&gt; '&lt;Значение поля&gt;',

где &lt;Имя поля&gt; - имя поля сообщения, по которому производится фильтрация, &lt;Оператор сравнения&gt; - операторы сравнения, как следует из их названия, позволяют сравнивать поле сообщения и заданное значение, &lt;Значение поля&gt; - значение для сравнения с полем сообщения.

Количество простых выражений может составить составное, например: (&lt;Простое выражение 1&gt; &amp; &lt;Простое выражение 2&gt;) &#x7C; &lt;Простое выражение 3&gt;,

где "&amp;" - логический оператор И, "&#x7C;" - оператор логического ИЛИ

В настоящее время допустимы следующие значения имени поля (&lt;Имя поля&gt;):

"To" - представляет поле TO сообщения, "Text" - представляет строку в заголовке или теле сообщения , "Bcc" - представляет поле BCC сообщения, "Body" - представляет строку в теле сообщения, "Cc" - представляет поле CC сообщения, "From" - представляет поле From сообщения, "Subject" - представляет строку в теме сообщения, "InternalDate" - внутренняя дата сообщения, "SentDate" - дата отправки сообщения

Кроме того, для IMAP-протокола разрешены следующие имена полей:

"Ответил" - представляет флаг сообщения /Ответил "Seen" - представляет флаг /Seen сообщения "Flagged" - представляет флаг /Flagged сообщения ge "Черновик" - представляет флаг /Черновик сообщения "Удалено" - представляет флаг Удалить/ сообщения "Последнее" - представляет Удалено/ флаг сообщения "MessageSize" - представляет размер (в байтах) сообщения

Кроме того, для Exchange разрешены следующие имена полей:

"IsRead" - Указывает, было ли сообщение прочитано "HasAttachment " - Указывает, есть ли в сообщении вложения "IsSubmitted" - Указывает, было ли сообщение отправлено в папку "Исходящие" "ContentClass" - представляет класс содержимого элемента

Кроме того, для файлов pst/ost разрешены следующие имена полей:

"MessageClass" - представляет класс сообщения "ContainerClass" - представляет класс контейнера папки "Importance" - представляет важность сообщения "MessageSize" - представляет размер (в байтах) сообщения "FolderName" - представляет имя папки "ContentsCount" - представляет общее количество элементов в папке «UnreadContentsCount» — представляет количество непрочитанных элементов в папке. "Подпапки" - указывает, есть ли в папке подпапки "Прочитано" - сообщение помечается как прочитанное "Имеет вложение" - сообщение имеет по крайней мере одно вложение "Неотправлено" - сообщение все еще составляется "Не изменено" - сообщение не было изменено с момента его первого сохранения (если оно не отправлено) или оно было доставлено (если отправлено) "FromMe" - пользователь, получивший сообщение, также был пользователем, отправившим сообщение "Повторно отправить" - сообщение содержит запрос на повторную отправку работа с отчетом о недоставке "NotifyRead" - пользователь, отправивший сообщение, запросил уведомление при первом его прочтении получателем "NotifyUnread" - пользователь, отправивший сообщение запросил уведомление, когда получатель удалит его до прочтения или срок действия объекта Message истечет "EverRead" - сообщение было прочитано хотя бы один раз

Значение поля (&lt;Значение поля&gt;) может принимать следующие значения:значения: Для текстовых полей - любая строка, Для полей типа даты - строка формата "д-МММ-гггг", напр. "10-Фев-2009", Для флагов (поля логического типа) - либо "Истина", либо "Ложь"

### Примеры

```csharp
MailQuery mailQuery = new MailQuery("(('From' Contains 'test@test.com' | 'Seen' = 'True') & 'SentDate' >= '12-May-2010')");
```
