## EmailClient.SetSupportedEncryptionUnsafe method

Определяет используемые версии протоколов шифрования SSL/TLS. Этот метод небезопасен и устанавливает протоколы шифрования без каких-либо проверок совместимости. Использование[`SupportedEncryption`](../supportedencryption) свойство для безопасной установки только тех протоколов, которые определенно поддерживаются .net framework. Обратите внимание, если ваш текущий .net framework не поддерживает этот уровень безопасности, будет выдано исключение при попытке установить соединение с сервером.
