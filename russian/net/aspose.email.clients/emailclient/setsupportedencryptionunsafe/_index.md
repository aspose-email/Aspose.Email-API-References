## EmailClient.SetSupportedEncryptionUnsafe method

Определяет используемые версии протоколов шифрования SSL/TLS. Этот метод не является безопасным и устанавливает протоколы шифрования без каких-либо проверок совместимости. Используйте свойство[`SupportedEncryption`](../supportedencryption)для безопасной установки только тех протоколов, которые определенно поддерживаются .net framework. Обратите внимание, если ваш текущий .net framework не поддерживает этот уровень безопасности, при попытке установить соединение с сервером будет выброшено исключение.

public void SetSupportedEncryptionUnsafe(EncryptionProtocols value)
