## MessageFormattingFlags enumeration

Представляет возможные параметры, которые можно установить в процессе форматирования сообщения для различных выходных данных.

public enum MessageFormattingFlags

| Имя | Ценность | Описание |

| None | `0` | Ни один из параметров не установлен. |
