## ListFoldersReturnOptions enumeration

Параметры возврата для операции ListFolders Обратите внимание, что эти параметры поддерживаются в случае, если сервер поддерживает RFC 5258 "Расширения команд IMAP LIST" Подробнее см. https://tools.ietf.org/html/rfc5258

```csharp
[Flags]
public enum ListFoldersReturnOptions
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| None | `0` | Не определено |
| Subscribed | `1` | SUBSCRIBED - заставляет команду LIST возвращать состояние подписки для всех совпадающих имен почтовых ящиков. Атрибут "\Subscribed" ДОЛЖЕН поддерживаться и ДОЛЖЕН быть точно рассчитан, когда указана опция возврата SUBSCRIBED. Кроме того, все флаги почтовых ящиков ДОЛЖНЫ быть точно вычислены (это отличается от поведения команды LSUB). |
| Children | `2` | CHILDREN - запрашивает информацию о дочернем почтовом ящике. Эта опция ДОЛЖНА поддерживаться всеми серверами. |
