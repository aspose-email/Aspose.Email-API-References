## HeaderType.FollowupTo property

Используется в новостях Usenet, чтобы указать, что будущие обсуждения (= продолжение) статьи должны проходить в другом наборе групп новостей, а не в группе новостей, на которую ответили. Чаще всего используется, когда статья размещается в нескольких группах новостей, а дальнейшее обсуждение происходит только в одной из них. Определено в RFC 1036:2.2.3, не стандартизовано для использования в электронной почте.

public static HeaderType FollowupTo { get; }
