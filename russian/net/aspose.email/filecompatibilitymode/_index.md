## FileCompatibilityMode enumeration

Определяет внутренние преобразования, которые обязательно нужно делать при загрузке или сохранении сообщения. По умолчанию CR не заменяется на CRLF, Tnef-вложение не сохраняется.

```csharp
[Flags]
public enum FileCompatibilityMode
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| None | `0` | Значения не установлены. |
| AllowCROnly | `1` | Показывает необходимость принудительной замены разделителей строк на CRLF. |
| PreserveTnefAttachments | `2` | Управляет поведением при загрузке/сохранении вложения TNEF |
