## SmtpClient.SucceededSending event

Добавляет или удаляет подписчика для уведомлений об успешной отправке почтовых сообщений. Обратите внимание, что уведомления будут вызываться из других потоков, поэтому вам следует реализовать обработчик событий потокобезопасности. Также обратите внимание, что это событие является общим для экземпляра почтового клиента, в случае, если клиент работает в режиме множественного подключения, через это событие будет обработано сообщений со всех подключений. Таким образом, клиент должен проанализировать в своем коде сообщение из событий, чтобы понять, какое сообщение относится к какой операции.

```csharp
public event EventHandler<MailMessageEventArgs> SucceededSending;
```
