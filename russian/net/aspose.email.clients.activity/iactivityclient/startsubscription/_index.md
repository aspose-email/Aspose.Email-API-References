## IActivityClient.StartSubscription method

Запускает подписку на указанный тип контента. Если подписка на указанный тип контента уже существует, эта операция используется для: - обновления свойств активного вебхука - включения вебхука который был отключен из-за чрезмерного количества неудачных уведомлений - повторно включить веб-перехватчик с истекшим сроком действия, указав более позднюю или нулевую дату истечения срока действия - удалить веб-перехватчик

```csharp
public Subscription StartSubscription(string contentType, Webhook webhook)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| contentType | String | Указывает тип содержимого. |
| webhook | Webhook | Указывает настроенный вебхук |

### Возвращаемое значение

Подписка в текущем состоянии
