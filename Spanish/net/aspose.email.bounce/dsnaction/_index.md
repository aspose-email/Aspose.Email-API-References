## DSNAction enumeration

Indica la acción realizada por el Reporting-MTA como resultado de su intento de entregar el mensaje.

```csharp
public enum DSNAction
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| None | `0` | Ninguna de acciones. |
| Delayed | `1` | Indica que el MTA de informes hasta ahora no ha podido entregar o retransmitir el mensaje, pero seguirá intentándolo. Es posible que se emitan mensajes de notificación adicionales a medida que el mensaje se retrase o se entregue con éxito, o si los intentos de entrega se abandonan posteriormente. |
| Delivered | `2` | Indica que el mensaje se entregó con éxito a la dirección del destinatario especificada por el remitente, que incluye "entrega" a una lista desplegable de correo. No indica que el mensaje haya sido leído. Este es un estado terminal y no se debe esperar más DSN para este destinatario. |
| Expanded | `3` | Indica que el mensaje se entregó correctamente a la dirección del destinatario según lo especificado por el remitente y que el MTA de informes lo reenvió más allá de ese destino a varias direcciones de destinatarios adicionales. Un valor de acción de "expandido" difiere de Entregado en que "expandido" no es un estado terminal. Es posible que se proporcionen más notificaciones "fallidas" y/o "retrasadas". |
| Failed | `4` | Indica que el mensaje no se pudo entregar al destinatario. El MTA de informes ha abandonado cualquier intento de entregar el mensaje a este destinatario. No se deben esperar más notificaciones. |
| Relayed | `5` | Indica que el mensaje se ha retransmitido o enviado a un entorno que no acepta la responsabilidad de generar DSN tras una entrega satisfactoria. Este valor de acción NO DEBE usarse a menos que el remitente haya solicitado una notificación de entrega exitosa para este destinatario. |
