## ListFoldersReturnOptions enumeration

Opciones de devolución para la operación ListFolders Tenga en cuenta que estas opciones son compatibles en caso de que el servidor admita RFC 5258 "Extensiones de comando IMAP LIST" Consulte más detalles en https://tools.ietf.org/html/rfc5258

```csharp
[Flags]
public enum ListFoldersReturnOptions
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| None | `0` | No definido |
| Subscribed | `1` | SUBSCRIBED: hace que el comando LIST devuelva el estado de suscripción para todos los nombres de buzón coincidentes. El atributo "\Suscrito" DEBE admitirse y DEBE calcularse con precisión cuando se especifica la opción de retorno SUSCRITO. Además, todos los indicadores de buzón DEBEN calcularse con precisión (esto difiere del comportamiento del comando LSUB). |
| Children | `2` | CHILDREN - solicita información del niño del buzón. Esta opción DEBE ser compatible con todos los servidores. |
