## IActivityClient.StopSubscription method

Esta operación detiene una suscripción al tipo de contenido especificado. Cuando se detiene una suscripción, ya no recibirá notificaciones y no podrá recuperar el contenido disponible. Si la suscripción se reinicia más tarde, tendrá acceso a nuevos contenido a partir de ese momento. No podrá recuperar el contenido que estaba disponible entre el momento en que se detuvo y se reinició la suscripción.

```csharp
public void StopSubscription(string contentType)
```

| --- | --- | --- |
| contentType | String | Indica el tipo de contenido. |
