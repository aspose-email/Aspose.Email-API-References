## MapiMessageItemBase.BodyRtf property

Obtiene o establece el texto del mensaje con formato RTF.

```csharp
public string BodyRtf { get; set; }
```

### El valor de la propiedad

La cadena que representa el cuerpo del mensaje rtf.

### Observaciones

Al establecer un valor, se actualizan los valores de las propiedades PR_RTF_COMPRESSED, PR_RTF_DECOMPRESSES, PR_BODY. Un valor de cadena que se establezca debe tener formato RTF. Por lo tanto, si es necesario establecer un valor en formato HTML, el valor debe codificarse primero dentro de RTF, según RTF Extensions Specification. Para establecer rápidamente el contenido del mensaje del cuerpo en formato HTML o Texto sin formato, utilice el método SetBodyContent. Al establecer un valor nulo o una cadena vacía, los valores de las propiedades BodyRtf y Body se establecen como nulos.
