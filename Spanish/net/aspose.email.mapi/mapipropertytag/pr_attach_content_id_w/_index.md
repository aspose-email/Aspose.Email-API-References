## MapiPropertyTag.PR_ATTACH_CONTENT_ID_W field

Contiene el encabezado de identificación de contenido de un archivo adjunto de mensaje MIME. Esta propiedad se usa para compatibilidad con MHTML. Compilación UNICODE.

```csharp
public static readonly long PR_ATTACH_CONTENT_ID_W;
```

### Observaciones

Se puede usar para distinguir entre archivos adjuntos en línea como imágenes de firmas y archivos adjuntos normales. Si el archivo adjunto tiene esta propiedad, entonces es una imagen incrustada.
