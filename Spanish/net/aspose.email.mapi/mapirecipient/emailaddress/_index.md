## MapiRecipient.EmailAddress property

Obtiene o establece la dirección de correo electrónico del destinatario o remitente del mensaje .

```csharp
public string EmailAddress { get; set; }
```

### El valor de la propiedad

La dirección de correo electrónico.

### Excepciones

| excepción | condición |
| --- | --- |
| ArgumentNullException | se lanza si la dirección del destinatario es nula o está vacía. |
| ArgumentException | se lanza si la dirección del destinatario no está en un formato reconocido. |

### Observaciones

Tenga en cuenta que esta propiedad muestra una dirección en formato SMTP. Significa que si MapiRecipient.AddressType == "EX" y MapiPropertyTag.PR_EMAIL_ADDRESS contienen una dirección en estilo Exchange, el valor para el relleno de la propiedad mapiRecipient.EmailAddress se selecciona de PR_SMTP_ADDRESS. Al establecer un valor, los valores de PR_SEARCH_KEY Las propiedades y PR_ENTRYID también se actualizan.
