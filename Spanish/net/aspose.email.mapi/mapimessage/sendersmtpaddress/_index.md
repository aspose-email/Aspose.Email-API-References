## MapiMessage.SenderSmtpAddress property

Obtiene o establece la dirección de correo electrónico del remitente del mensaje.

```csharp
public string SenderSmtpAddress { get; set; }
```

### El valor de la propiedad

La cadena que representa la dirección de correo electrónico del remitente.

### Excepciones

| excepción | condición |
| --- | --- |
| ArgumentException | se lanza si la dirección del remitente no está en un formato reconocido. |

### Observaciones

Al establecer un valor, los valores de las propiedades PR_SENDER_SEARCH_KEY y PR_SENDER_ENTRYID también se actualizan. Al establecer un valor nulo o una cadena vacía, los valores de las propiedades se establecen como nulos.
