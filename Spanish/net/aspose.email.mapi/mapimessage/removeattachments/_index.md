## MapiMessage.RemoveAttachments method

Elimina todos los archivos adjuntos de los archivos de mensajes de Outlook especificados.


| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| path | String | El nombre del archivo de mensajes de Outlook. |

### Valor_devuelto

La colección de adjuntos.

### Ejemplos

El siguiente ejemplo muestra cómo destruir archivos adjuntos en archivos de mensajes de Outlook.

[C#]

[Visual Basic]

```csharp
//Eliminar archivos adjuntos de los archivos de mensajes de Outlook
MapiAttachmentCollection attachments = MapiMessage.RemoveAttachments(@"c:\outlookmessage.msg");

//Archivos adjuntos
foreach(MapiAttachment att in attachments)
{
   Console.WriteLine("Attachment Name:"+att.FileName);
   att.Save(att.FileName);
}
```
