## ResolveRecipientsRequest.To property

Especifica uno o más destinatarios para resolver. El valor puede tener hasta 256 caracteres de longitud. El resultado de incluir más de 1000 elementos Para en la solicitud no está definido. El servidor PUEDE devolver un error de estado de protocolo en respuesta a una solicitud de comando de este tipo. Si la solicitud de comando ResolveRecipients incluye la Disponibilidad, se puede incluir en la solicitud un máximo de 100 elementos Para que contengan direcciones SMTP. Si se incluyen más de 100 direcciones SMTP en la solicitud, se devuelve el valor de Estado 160 en la respuesta. Si la solicitud del comando ResolveRecipients incluye Disponibilidad e incluye un elemento Para para un usuario ambiguo, la respuesta no incluye un elemento MergedFreeBusy para ese usuario Solo los usuarios o las listas de distribución especificados con direcciones SMTP válidas o una cadena identificable de forma única en el mensaje de solicitud Para elemento tienen elementos MergedFreeBusy incluidos en la respuesta. Si la solicitud del comando ResolveRecipients incluye la Disponibilidad y el elemento Para especifica un grupo de distribución, entonces el los datos de disponibilidad se devuelven como una sola cadena que combina los datos de los miembros individuales del grupo de distribución. Si el grupo de distribución contiene más de 20 miembros, se devuelve un valor de elemento de estado de 161 en la respuesta que indica que la información de disponibilidad combinada de un grupo de distribución tan grande no es útil.
