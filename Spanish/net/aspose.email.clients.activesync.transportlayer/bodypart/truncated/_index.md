## BodyPart.Truncated property

Especifica si el cuerpo del elemento se ha truncado de acuerdo con el elemento BodyPreference indicado por el cliente. Si el valor es TRUE, entonces el cuerpo del elemento se ha truncado. Si el valor es FALSO o la propiedad Truncado no está configurada, entonces el cuerpo del elemento no se ha truncado.

public bool? Truncated { get; set; }
