## SyncRequest.Wait property

Especifica la cantidad de minutos que el servidor DEBERÍA demorar una respuesta si no se agregan elementos nuevos a las carpetas incluidas. Los valores válidos para la espera son de 1 a 59. Cuando el cliente solicita un intervalo de espera que está fuera del rango aceptable, el servidor enviará una respuesta que incluye un valor de Estado de 14 y un Límite. Se puede incluir Espera o Intervalo de latido en una solicitud de comando de sincronización, pero no ambos. Si se incluyen ambos, la respuesta del servidor contendrá un valor de Estado de 4.
