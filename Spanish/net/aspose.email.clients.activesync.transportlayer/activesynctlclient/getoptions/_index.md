## ActiveSyncTLClient.GetOptions method

El método estático GetOptions se usa para descubrir qué versiones de protocolo se admiten y qué comandos de protocolo se admiten en el servidor. El cliente usa el método estático GetOptions para determinar si el servidor admite las mismas versiones del protocolo que admite el cliente.

```csharp
public static ASProtocolVersions GetOptions(string uri, NetworkCredential credentials, 
    out string[] supportedServerCommands, out string[] supportedServerProtocols)
```
