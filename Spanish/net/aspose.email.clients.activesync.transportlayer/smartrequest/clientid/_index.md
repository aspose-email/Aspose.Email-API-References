## SmartRequest.ClientId property

Especifica el ID de mensaje único (MID) del cliente. El valor de ClientId puede tener hasta 40 caracteres de longitud y DEBE ser único para cada mensaje, ya que el servidor usará el valor de ClientId para identificar mensajes duplicados. El valor de ClientId puede ser un simple contador incrementado para cada nuevo mensaje.

public string ClientId { get; set; }
