## SmartRequest.Mime property

Contiene el mensaje codificado en MIME. El contenido Mime se transfiere como un BLOB opaco dentro de las etiquetas WBXML, como se especifica en [WBXML1.2]. Si el mensaje contiene una solicitud de reunión, el elemento Mime contiene los detalles de la reunión en formato iCalendar [MS-OXCICAL] o formato de encapsulación de transporte neutral (TNEF) [MS-OXTNEF]. Como se especifica en [RFC2447] sección 3.4, las solicitudes de reunión de iCalendar tienen un tipo de contenido de "texto/calendario" con el parámetro de método establecido en "SOLICITUD".

public string Mime { get; set; }
