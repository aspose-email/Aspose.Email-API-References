## SmartRequest.ReplaceMime property

Especifica si el cliente está enviando el mensaje completo. Cuando ReplaceMime es VERDADERO, el servidor NO DEBE incluir el cuerpo o los archivos adjuntos del mensaje original que se está reenviando. Cuando ReplaceMime es FALSO, el cliente DEBE agregar el cuerpo del mensaje original como archivos adjuntos al mensaje saliente. El cliente puede usar esta propiedad para indicar si el mensaje se editó en línea o si el mensaje tenía texto de respuesta/reenvío antepuesto al mensaje de origen. Si ReplaceMime es VERDADERO, el mensaje fue editado.

public bool ReplaceMime { get; set; }
