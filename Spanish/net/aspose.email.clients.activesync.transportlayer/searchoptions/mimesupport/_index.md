## SearchOptions.MIMESupport property

Habilita la compatibilidad con MIME para elementos de correo electrónico que se envían desde el servidor al cliente. Si el elemento airsync:MIMESupport se establece en 'SendForSecureMIMEonly' (1) o 'SendForAll' (2) en la solicitud de búsqueda: - La propiedad de airsyncbase:BodyPreference, el Tipo, DEBE incluirse en la solicitud de búsqueda, que contiene un valor de 'MIME' (4) para informar al servidor que el dispositivo puede leer el MIME BLOB. - La respuesta del servidor DEBE incluir el airsyncbase:Body, que es un elemento secundario de las Propiedades. El airsyncbase:Body DEBE contener las siguientes propiedades en una respuesta de búsqueda S/MIME: - El airsyncbase:Type con un valor de 'MIME' (4) para informar al dispositivo que los datos son un MIME BLOB. - El airsyncbase :EstimatedDataSize para especificar el tamaño total aproximado de los datos. - Airsyncbase:Truncated para indicar si el MIME BLOB está truncado. - Airsyncbase:Data que contiene el MIME BLOB completo.
