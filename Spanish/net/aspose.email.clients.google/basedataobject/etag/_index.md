## BaseDataObject.ETag property

Una ETag o etiqueta de entidad es uno de varios mecanismos que proporciona HTTP para la validación de caché web y que permite a un cliente realizar solicitudes condicionales. Esto permite que los cachés sean más eficientes y ahorra ancho de banda, ya que un servidor web no necesita enviar una respuesta completa si el contenido no ha cambiado. Las etiquetas ETag también se pueden usar para el control de concurrencia optimista, como una forma de ayudar a evitar que las actualizaciones simultáneas de un recurso se sobrescriban entre sí.
