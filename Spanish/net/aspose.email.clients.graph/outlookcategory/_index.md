## OutlookCategory class

Representa una categoría por la que un usuario puede agrupar elementos de Outlook como mensajes y eventos. El usuario define categorías en una lista maestra y puede aplicar una o más de estas categorías definidas por el usuario a un elemento. https://docs.microsoft.com/en-us/graph/api/resources/outlookcategory?view =grafo-resto-1.0
