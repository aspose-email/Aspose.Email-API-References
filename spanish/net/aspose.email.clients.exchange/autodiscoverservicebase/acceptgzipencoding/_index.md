## AutodiscoverServiceBase.AcceptGzipEncoding property

Obtiene o establece un valor que indica si se debe aceptar la codificación de compresión GZip.

```csharp
public bool AcceptGzipEncoding { get; set; }
```

### Observaciones

Este valor le indicará al servidor que el cliente puede manejar la codificación de compresión GZip. El servidor solo enviará contenido comprimido Gzip si ha sido configurado para hacerlo.
