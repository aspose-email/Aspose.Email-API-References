## ImapFolderInfo.HasChildren property

Obtiene el valor que indica si la carpeta contiene subcarpetas. Esta opción solo es accesible en caso de que el servidor admita las extensiones de comando IMAP4 LIST (rfc5258) Ver más: http://tools.ietf.org/html/rfc5258 Si el valor es`verdadero` , indica que la carpeta tiene subcarpetas secundarias a las que puede acceder el usuario autenticado actualmente; de lo contrario, false.
