## ImapSpecialFolderTypes enumeration

Representa la enumeración de buzones de correo de uso especial Más detalles ver en RFC6154 http://tools.ietf.org/html/rfc6154

```csharp
public enum ImapSpecialFolderTypes
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| NotSpecified | `0` | Si se admite la extensión IMAP LIST para buzones de correo de uso especial (RFC6154), significa que el buzón es normal y no se utiliza para fines especiales. |
| All | `1` | Este buzón presenta todos los mensajes en el almacén de mensajes del usuario. Las implementaciones PUEDEN omitir algunos mensajes, como, quizás, los de \Trash y \Junk. Cuando se admite este uso especial, es casi seguro que represente un buzón virtual. |
| Archive | `2` | Este buzón se utiliza para archivar mensajes. El significado de un buzón de "archivo" depende del servidor; por lo general, se usará para sacar los mensajes de la bandeja de entrada, o mantenerlos fuera del camino del usuario, sin dejar de hacerlos accesibles. |
| Drafts | `3` | Este buzón se usa para guardar borradores de mensajes, mensajes que se están redactando pero que aún no se han enviado. En algunas implementaciones de servidor, este puede ser un buzón virtual que contiene mensajes de otros buzones que están marcados con "\Borrador" bandera de mensaje Alternativamente, esto podría ser solo un consejo para que un cliente coloque borradores aquí. |
| Flagged | `4` | Este buzón presenta todos los mensajes marcados de alguna manera como "importantes". Cuando se admite este uso especial, es probable que represente un buzón virtual que recopila mensajes (de otros buzones) que están marcados con el indicador de mensaje "\Marcado". |
| Junk | `5` | Este buzón es donde se guardan los mensajes que se consideran correo no deseado. Algunas implementaciones de servidor pueden colocar mensajes aquí automáticamente. Alternativamente, esto podría ser solo un consejo para un filtro de spam del lado del cliente. |
| Sent | `6` | Este buzón se utiliza para almacenar copias de los mensajes que se han enviado. Algunas implementaciones de servidor pueden colocar mensajes aquí automáticamente. Alternativamente, esto podría ser solo un consejo para que un cliente guarde los mensajes enviados aquí. |
| Trash | `7` | Este buzón se utiliza para almacenar mensajes que se han eliminado o marcado para su eliminación. En algunas implementaciones de servidor, este puede ser un buzón virtual que contiene mensajes de otros buzones que están marcados con el indicador de mensaje "\Deleted". Alternativamente, esto podría ser solo un consejo de que un cliente que elige no usar el modelo IMAP "\Deleted" debería usar esto como su ubicación de basura. En implementaciones de servidor que esperan estrictamente el modelo IMAP "\Deleted", es probable que este uso especial no sea compatible. |
| Important | `8` | Este buzón se usa para contener mensajes que se han marcado como importantes. El atributo de buzón "\Important" es una señal de que el buzón contiene mensajes que probablemente sean importantes para el usuario. En una implementación que también admita "$Important ", es probable que este uso especial represente un buzón virtual que recopila mensajes (de otros buzones) que están marcados con la palabra clave "$Importante". En otras implementaciones, el sistema puede colocar automáticamente mensajes allí en función de los mismos tipos de heurística que se indican para la palabra clave "$Importante". Las distinciones entre "\Importante" y "\Marcado" para los buzones de correo son similares a las que existen entre "$Importante" y "\Marcado" para los mensajes. https://tools.ietf.org/html/rfc8457 |
