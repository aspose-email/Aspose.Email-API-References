## RestoreSettings.NumberOfAttemptsToRrepeat property

Obtiene o establece el valor que define el número de intentos para repetir la operación fallida En caso de que algún comando IMAP dentro de la operación de respaldo devuelva un resultado fallido, el cliente IMAP intenta repetir esta operación nuevamente de acuerdo con la cantidad definida de veces. Por ejemplo, si la operación FETCH devuelve el error AE_1_1_0243 FETCH 219 (BODY) AE_1_1_0243 NO[NO DISPONIBLE] El servicio FETCH no está disponible temporalmente El cliente intenta ejecutarlo de nuevo.

public int NumberOfAttemptsToRrepeat { get; set; }
