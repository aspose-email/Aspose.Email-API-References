## HeaderType.DispositionNotificationTo property

Cuando se establece el campo DispositionNotificationTo, se realiza una solicitud de MDN (Notificación de entrega de mensaje). El software de correo electrónico del destinatario (Outlook, Eudora, etc.) puede ignorar silenciosamente la solicitud, o puede solicitar permiso al usuario para enviar el MDN. No hay garantía del "acuse de recibo". El campo DispositionNotificationTo es el estándar de facto para solicitar acuses de recibo (es decir, MDN o notificaciones de entrega de mensajes).
