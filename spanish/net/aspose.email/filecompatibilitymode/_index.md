## FileCompatibilityMode enumeration

Define conversiones internas, que necesariamente se deben realizar al cargar o guardar un mensaje. Por defecto, CR no se reemplaza por CRLF, El archivo adjunto Tnef no se guarda.

```csharp
[Flags]
public enum FileCompatibilityMode
```

| Nombre | Valor | Descripción |
| --- | --- | --- |
| None | `0` | No se establecen valores. |
| AllowCROnly | `1` | Muestra la necesidad del reemplazo forzado de los separadores de línea por CRLF. |
| PreserveTnefAttachments | `2` | Controla el comportamiento de carga/guardado de archivos adjuntos TNEF |
