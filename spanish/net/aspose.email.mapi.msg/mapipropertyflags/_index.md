## MapiPropertyFlags enumeration

Representa indicadores que se pueden establecer en una propiedad MAPI.

```csharp
[Flags]
public enum MapiPropertyFlags
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| PROPATTR_MANDATORY | `1` | Si se establece este indicador para una propiedad, esa propiedad NO DEBE eliminarse del archivo .msg (independientemente del almacenamiento en el que se encuentre) y las implementaciones DEBEN devolver un error si se intenta hacerlo. Este indicador se establece en circunstancias en las que la implementación depende de que esa propiedad siempre esté presente en el archivo .msg una vez que se escribe allí. |
| PROPATTR_READABLE | `2` | Si este indicador no está establecido en una propiedad, esa propiedad NO DEBE leerse del archivo .msg y las implementaciones DEBEN devolver un error si se intenta leerla. Este indicador se establece en todas las propiedades a menos que haya una razón específica de implementación para evitar que una propiedad se lea desde el archivo .msg. |
| PROPATTR_WRITABLE | `4` | Si este indicador no se establece en una propiedad, esa propiedad NO DEBE modificarse ni eliminarse y las implementaciones DEBEN devolver un error si se intenta hacerlo. Este indicador se establece en circunstancias en las que la implementación depende de que las propiedades se puedan escribir. |
