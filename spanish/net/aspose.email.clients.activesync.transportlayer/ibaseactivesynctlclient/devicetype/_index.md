## IBaseActiveSyncTLClient.DeviceType property

El tipo de dispositivo se especifica mediante la parte de la regla ABNF de especificación de tipo de dispositivo del valor de consulta de texto sin formato. El valor, representado por la regla ABNF de tipo de dispositivo, es cualquier cadena que especifica un tipo de dispositivo. "SP" especifica un SmartPhone y "PPC" especifica un PocketPC. Otros dispositivos cliente envían cadenas únicas para su tipo de dispositivo específico. Cada solicitud de un dispositivo cliente DEBE incluir la misma cadena de tipo de dispositivo.

public string DeviceType { get; set; }
