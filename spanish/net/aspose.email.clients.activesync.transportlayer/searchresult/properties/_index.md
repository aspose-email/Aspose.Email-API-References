## SearchResult.Properties property

Las propiedades de respuesta del comando de búsqueda son un contenedor de propiedades que se aplican a una entrada individual que coincide con la cadena de búsqueda del elemento de consulta. Por ejemplo, el elemento Propiedades contiene un elemento para cada propiedad GAL no vacía con valor de texto que se adjunta a la entrada GAL coincidente. Solo se devuelven aquellas propiedades que están adjuntas a la entrada GAL específica; por lo tanto, se pueden devolver diferentes conjuntos de propiedades en el XML de respuesta para diferentes entradas de GAL coincidentes. Cada elemento en el contenedor de propiedades tiene como ámbito el espacio de nombres apropiado que se especifica en el elemento de búsqueda de nivel superior.
