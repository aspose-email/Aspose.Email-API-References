## SearchOptions class

Contiene las opciones de búsqueda. El nombre de usuario y la contraseña solo se pueden enviar en la solicitud después de recibir un valor de estado de 14. El servidor requiere estas credenciales para acceder a los recursos solicitados. El cliente DEBE enviarlos únicamente a través de una conexión segura o confiable, y solo en respuesta a un valor de Estado de 14. Las opciones admitidas varían según la tienda que se busca. La siguiente tabla enumera las opciones válidas para cada tienda. GAL: rango, nombre de usuario, contraseña, imagen Buzón: rango, recorrido profundo, resultados de reconstrucción, BodyPreference, BodyPartPreference, RightsManagementSupport DocumentLibrary: rango, nombre de usuario, contraseña BodyPartPreference solo es válido en la búsqueda solicitudes de comando que incluyen un ConversationId.
