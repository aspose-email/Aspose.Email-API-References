## RightsManagementLicense.ProgrammaticAccessAllowed property

Especifica si las aplicaciones de terceros pueden acceder mediante programación al contenido del mensaje de correo electrónico. El valor es TRUE si las aplicaciones de terceros pueden acceder al contenido del mensaje de correo electrónico mediante programación; en caso contrario, FALSO. Un valor de TRUE indica si otras aplicaciones pueden acceder al contenido protegido. El contenido protegido consta del cuerpo del mensaje y los archivos adjuntos.

public bool ProgrammaticAccessAllowed { get; set; }
