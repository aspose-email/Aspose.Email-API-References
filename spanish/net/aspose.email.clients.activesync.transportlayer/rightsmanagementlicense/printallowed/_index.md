## RightsManagementLicense.PrintAllowed property

Especifica si el usuario puede imprimir el correo electrónico. Este elemento no indica el soporte del cliente para imprimir un mensaje de correo electrónico; solo especifica si el mensaje de correo electrónico se puede imprimir si el cliente admite la impresión. El valor es VERDADERO si el usuario puede imprimir el correo electrónico; en caso contrario, FALSO.

public bool PrintAllowed { get; set; }
