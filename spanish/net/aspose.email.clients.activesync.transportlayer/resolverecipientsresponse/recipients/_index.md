## ResolveRecipientsResponse.Recipients property

Representa destinatarios que se han resuelto. Se devuelve más de un destinatario en una respuesta si el Para especificado en la solicitud se resolvió en una lista de distribución o se encontró que era ambiguo. El código de estado devuelto en la respuesta se puede usar para determinar si se encontró que el destinatario era ambiguo. El destinatario sería una coincidencia sugerida si se determina que el destinatario especificado en la solicitud es ambiguo. Los certificados se devuelven si el cliente solicitó que se devolvieran los certificados en la respuesta.
