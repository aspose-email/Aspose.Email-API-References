## SyncCollectionRequest.DeletesAsMoves property

Solicitudes que indican que los elementos eliminados DEBERÍAN moverse a la carpeta Elementos eliminados. Si el elemento DeletesAsMoves se establece en falso, la eliminación es permanente. Si el cliente desea eliminar elementos de forma permanente, la solicitud DEBE incluir el elemento DeletesAsMoves con un valor FALSO. Un valor de VERDADERO, que es el valor predeterminado, indica que los elementos eliminados se mueven a la carpeta Elementos eliminados.

public bool? DeletesAsMoves { get; set; }
