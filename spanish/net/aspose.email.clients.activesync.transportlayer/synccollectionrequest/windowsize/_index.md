## SyncCollectionRequest.WindowSize property

Especifica un número máximo de elementos modificados en una colección o una solicitud que DEBERÍA incluirse en la respuesta de sincronización. Si no se define WindowSize, el servidor se comporta como si se enviara un elemento WindowSize con un valor de 100. El servidor interpreta el valor 0 (cero) y los valores superiores a 512 como 512.
