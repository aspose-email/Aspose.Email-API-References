## AccountInformation.SendDisabled property

Especifica si el cliente puede enviar mensajes usando la cuenta dada. El valor de SendDisabled es VERDADERO si el cliente no puede enviar mensajes usando la cuenta dada; de lo contrario, FALSE. SendDisabled no se admite cuando la versión del protocolo es 12.1 o 14.0.

public bool? SendDisabled { get; set; }
