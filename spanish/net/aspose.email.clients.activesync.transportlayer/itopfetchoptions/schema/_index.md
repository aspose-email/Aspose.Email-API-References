## ItOpFetchOptions.Schema property

Especifica el esquema del elemento que se recuperará. El esquema se admite dentro de las opciones para las solicitudes de recuperación de PIM (sección 2.2.3.63.1). No se admite cuando el cliente está recuperando elementos de una biblioteca de documentos o recuperando un archivo adjunto. Si no se especifica el elemento Esquema, el servidor permite que se recuperen todas las propiedades.
