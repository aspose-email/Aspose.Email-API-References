## ItOpFetchOptions.Range property

Especifica el rango de bytes que el cliente puede recibir en respuesta a la operación Fetch para un elemento de la biblioteca de documentos. El elemento Rango es un campo opcional. El elemento Rango contiene un valor de cadena en el formato MN, donde M es menor que N, M es el valor mínimo y N es el valor máximo. El rango de bytes está indexado a cero; el primer byte se indica con un 0 (cero). Si se omite el rango en la solicitud de obtención, se obtiene el elemento completo. Si la referencia de archivo está presente en la solicitud, el elemento de rango es el único elemento secundario válido de el elemento Opciones.
