## IActiveSyncTLClient.MoveItem method

El comando MoveItems mueve un elemento o elementos de una carpeta del servidor a otra.

```csharp
public MoveItemResponse MoveItem(string srcMsgId, string srcFldId, string dstFldId)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| srcMsgId | String | Especifica el ID de servidor del elemento que se movió. El valor del elemento SrcMsgId puede tener hasta 64 caracteres de longitud. |
| srcFldId | String | Especifica el ID del servidor de la carpeta de origen (es decir, la carpeta que contiene los elementos que se van a mover). El valor del elemento SrcFldId puede tener hasta 64 caracteres de longitud |
| dstFldId | String | Especifica el ID de servidor de la carpeta de destino (es decir, la carpeta a la que se mueven los elementos). El valor de la propiedad DstFldId puede tener hasta 64 caracteres de longitud. |
