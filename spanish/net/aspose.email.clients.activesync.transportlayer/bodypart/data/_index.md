## BodyPart.Data property

El contenido del elemento de datos es una cadena en el formato especificado por la propiedad Tipo. Si el valor del Tipo es RTF, el valor del elemento Datos se codifica utilizando la codificación base64. Si la propiedad Truncado se establece en verdadero, los datos del elemento Datos se truncan. La propiedad EstimatedDataSize proporciona una estimación aproximada del tamaño real del contenido completo de la cadena de datos.
