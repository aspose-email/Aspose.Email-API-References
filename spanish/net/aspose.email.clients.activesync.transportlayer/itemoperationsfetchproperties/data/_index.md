## ItemOperationsFetchProperties.Data property

Contiene el contenido del elemento para las respuestas de contenido en línea. El contenido del elemento de datos es una codificación base64 del documento binario, el archivo adjunto o los datos del cuerpo. La propiedad Rango indica el tamaño de los datos (en bytes) que se encuentran dentro del elemento Datos. El tamaño total del elemento (en bytes) se indica mediante el elemento Total.
