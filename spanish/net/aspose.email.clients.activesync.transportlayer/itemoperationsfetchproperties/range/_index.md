## ItemOperationsFetchProperties.Range property

Especifica el rango de bytes que el cliente puede recibir en respuesta a la operación Fetch para un elemento de la biblioteca de documentos. El elemento Range contiene un valor de cadena con el formato mn, donde m es menor que n, y m es el valor mínimo y n es el valor máximo. El rango de bytes está indexado a cero; el primer byte se indica con un 0 (cero). Si se omite el elemento Rango en la solicitud de obtención, se obtiene el elemento completo.

public string Range { get; set; }
