## EASProvisionDoc.MinDevicePasswordLength property

Especifica la longitud mínima de la contraseña del cliente. MinDevicePasswordLength puede estar vacío o tener un valor no inferior a 1 ni superior a 16. Si la propiedad es nula o el valor de esta propiedad es 1, no existe una longitud mínima para la contraseña del dispositivo. Si el valor de DevicePasswordEnabled es FALSO, el cliente DEBERÍA ignorar esta propiedad.

public int? MinDevicePasswordLength { get; set; }
