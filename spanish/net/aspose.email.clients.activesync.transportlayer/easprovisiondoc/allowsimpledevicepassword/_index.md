## EASProvisionDoc.AllowSimpleDevicePassword property

Especifica si el cliente permite contraseñas simples. Una contraseña simple consiste solo en caracteres repetidos ("2222") o secuenciales ("abcd"). Si AllowSimpleDevicePassword es nulo, un cliente DEBE tratar este valor como VERDADERO. Si el valor de DevicePasswordEnabled se establece en FALSO , el cliente DEBERÍA ignorar esta propiedad.
