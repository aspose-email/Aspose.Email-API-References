## EASProvisionDoc.PasswordRecoveryEnabled property

Especifica si el servidor admite el almacenamiento de una contraseña de recuperación para que la envíe el cliente mediante el comando Configuración. Una contraseña de recuperación es una contraseña especial creada por el cliente que otorga al administrador o al usuario la capacidad de iniciar sesión en el dispositivo una vez , después de lo cual se requiere que el usuario cree una nueva contraseña. Luego, el cliente crea una nueva contraseña de recuperación. Si PasswordRecoveryEnabled se establece en TRUE, el servidor admite el almacenamiento de una contraseña de recuperación enviada por el dispositivo. Si la propiedad se establece en FALSO, el dispositivo NO DEBE enviar una contraseña de recuperación porque el servidor no admite el almacenamiento de la contraseña. Si PasswordRecoveryEnabled es nulo, un cliente DEBERÍA tratar este valor como FALSO. Si el valor de DevicePasswordEnabled es FALSO, el cliente DEBE ignorar esta propiedad. Esta propiedad DEBE ignorarse si el cliente no admite contraseñas de recuperación.

public bool? PasswordRecoveryEnabled { get; set; }
