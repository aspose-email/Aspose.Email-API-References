## EASProvisionDoc.AlphanumericDevicePasswordRequired property

Especifica si un cliente requiere una contraseña alfanumérica. Si AlphanumericDevicePasswordRequired es nulo, un cliente DEBE tratar este valor como FALSO. Si el valor de DevicePasswordEnabled es FALSO, el cliente DEBE ignorar esta propiedad.
