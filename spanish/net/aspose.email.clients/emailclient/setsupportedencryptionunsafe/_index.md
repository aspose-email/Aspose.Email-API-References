## EmailClient.SetSupportedEncryptionUnsafe method

Define las versiones de los protocolos de encriptación SSL/TLS que se utilizarán. Este método no es seguro y establece los protocolos de encriptación sin ninguna verificación de compatibilidad. Usar[`SupportedEncryption`](../supportedencryption) propiedad para configurar de forma segura solo los protocolos que definitivamente son compatibles con .net framework. Tenga en cuenta que si su actual .net framework no admite este nivel de seguridad, se generará una excepción al intentar establecer una conexión con el servidor.
