## EmailClient.SupportedEncryption property

Define las versiones de los protocolos de encriptación SSL/TLS que se utilizarán. POR FAVOR PRESTA ATENCIÓN, puedes configurar solo aquellas versiones de protocolo compatibles con .net framework. SI ALGUNAS VERSIONES DEL PROTOCOLO NO SON COMPATIBLES CON TU VERSIÓN ACTUAL DE .NET FRAMEWORK, SE IGNORARÁN Y OMITIRÁN. PUEDE LLEVAR A UNA DEGRADACIÓN DEL NIVEL DE SEGURIDAD DE TLS. ¡¡¡EN ESTE CASO NO SE GENERARÁ EXCEPCIÓN!!! Por favor, consulte[`EncryptionProtocols`](../../../aspose.email.clients.base/encryptionprotocols) documentación para obtener más detalles. Utilice[`SetSupportedEncryptionUnsafe`](../setsupportedencryptionunsafe) método si desea configurar los protocolos de cifrado sin ninguna verificación de compatibilidad. El valor predeterminado es: Tls &#x7C; Tls11 &#x7C; Tls12 &#x7C; Tls13 (en caso de que su versión actual de .NET Framework sea compatible con estas versiones de TLS)

```csharp
public virtual EncryptionProtocols SupportedEncryption { get; set; }
```
