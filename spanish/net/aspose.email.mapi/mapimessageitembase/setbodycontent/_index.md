## SetBodyContent(string, BodyContentType) {#setbodycontent}

Establece el contenido del cuerpo.

```csharp
public virtual void SetBodyContent(string content, BodyContentType contentType)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| content | String | El contenido. |
| contentType | BodyContentType | Tipo del contenido. |

### Excepciones

| excepción | condición |
| --- | --- |
| ArgumentNullException | lanza si la cadena de contenido especificada es nula o está vacía. |

### Observaciones

Se proporciona para configurar el contenido del mensaje del cuerpo en formatos RTF, HTML o Texto sin formato. Al establecer un valor, los valores de las propiedades PR_RTF_COMPRESSED, PR_RTF_DECOMPRESSES, PR_BODY también se actualizan. Tenga en cuenta que después de establecer el valor en formato HTML, la propiedad BodyRtf devuelve el valor que está codificado en RTF.
