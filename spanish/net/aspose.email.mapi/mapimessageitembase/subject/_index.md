## MapiMessageItemBase.Subject property

Obtiene o establece el asunto del mensaje.

```csharp
public string Subject { get; set; }
```

### El valor de la propiedad

La cadena que representa el asunto del mensaje.

### Observaciones

Al establecer un valor, los valores de las propiedades SubjectPrefix(PR_SUBJECT_PREFIX) y NormalizedSubject(PR_NORMALIZED_SUBJECT) también se actualizan. Si el Asunto no tiene prefijo, el valor de la propiedad AsuntoPrefijo se establece como nulo. Al establecer un valor nulo o una cadena vacía, los valores de las propiedades Asunto, AsuntoPrefijo, Asunto normalizado se establecen como nulos.
