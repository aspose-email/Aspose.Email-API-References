## MapiMessage.SentRepresentingEmailAddress property

Obtiene o establece la dirección de correo electrónico del usuario de mensajería representado por el remitente.

```csharp
public string SentRepresentingEmailAddress { get; set; }
```

### El valor de la propiedad

La cadena que representa el envío que representa la dirección de correo electrónico.

### Excepciones

| excepción | condición |
| --- | --- |
| ArgumentException | lanza si la dirección de correo electrónico no está en un formato reconocido. |

### Observaciones

Al establecer un valor, los valores de las propiedades PR_SENT_REPRESENTING_SEARCH_KEY y PR_SENT_REPRESENTING_ENTRYID también se actualizan. Al establecer un valor nulo o una cadena vacía, los valores de la propiedad se establecen como nulos.
