## AmpAttributes.Media property

Il valore di media è una media query. Se la query non corrisponde, il rendering dell'elemento non viene eseguito e le relative risorse e potenzialmente le risorse figlio non verranno recuperate. Se la finestra del browser cambia dimensione o orientamento, le media query vengono rivalutate e gli elementi vengono nascosti e mostrati in base ai nuovi risultati.
