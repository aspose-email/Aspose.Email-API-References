## MapiMessageItemBase.Subject property

Ottiene o imposta l'oggetto del messaggio.

```csharp
public string Subject { get; set; }
```

### Valore della proprietà

La stringa che rappresenta l'oggetto del messaggio.

### Osservazioni

Quando si imposta un valore, vengono aggiornati anche i valori delle proprietà SubjectPrefix(PR_SUBJECT_PREFIX) e NormalizedSubject(PR_NORMALIZED_SUBJECT). Se Oggetto non ha prefisso, il valore della proprietà SubjectPrefix è impostato su null. Quando si imposta un valore nullo o una stringa vuota, i valori delle proprietà Subject, SubjectPrefix, NormalizedSubject vengono impostati su null.
