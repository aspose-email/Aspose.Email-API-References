## MapiMessageItemBase.SetBodyRtf method

Ottiene o imposta il testo del messaggio in formato RTF.

```csharp
public void SetBodyRtf(string content, bool compression)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| content | String | Il contenuto. |
| compression | Boolean | Specificare che il contenuto deve essere compresso. |

### Osservazioni

Quando si imposta un valore, i valori delle proprietà PR_RTF_COMPRESSED, PR_RTF_DECOMPRESSES, PR_BODY vengono aggiornati. Un valore stringa da impostare deve avere il formato RTF. Pertanto, se è necessario impostare un valore in formato HTML, il valore deve essere prima codificato in RTF, secondo la specifica delle estensioni RTF. Per impostare rapidamente il contenuto del corpo del messaggio in formato HTML o Testo normale, utilizza il metodo SetBodyContent. Quando si imposta un valore nullo o una stringa vuota, i valori delle proprietà BodyRtf e Body vengono impostati su null.
