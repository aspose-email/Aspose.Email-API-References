## MapiRecipient.EmailAddress property

Ottiene o imposta l'indirizzo e-mail del destinatario o del mittente del messaggio .

```csharp
public string EmailAddress { get; set; }
```

### Valore della proprietà

L'indirizzo email.

### Eccezioni

| eccezione | condizione |
| --- | --- |
| ArgumentNullException | genera se l'indirizzo del destinatario è nullo o vuoto. |
| ArgumentException | genera se l'indirizzo del destinatario non è in un formato riconosciuto. |

### Osservazioni

Nota, questa proprietà visualizza un indirizzo in formato SMTP. Significa che se MapiRecipient.AddressType == "EX" e MapiPropertyTag.PR_EMAIL_ADDRESS contengono un indirizzo in stile Exchange, il valore per il riempimento della proprietà mapiRecipient.EmailAddress viene prelevato da PR_SMTP_ADDRESS. Quando si imposta un valore, i valori di PR_SEARCH_KEY Vengono aggiornate anche le proprietà e PR_ENTRYID.
