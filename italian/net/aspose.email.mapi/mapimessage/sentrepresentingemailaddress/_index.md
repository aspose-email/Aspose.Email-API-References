## MapiMessage.SentRepresentingEmailAddress property

```csharp
public string SentRepresentingEmailAddress { get; set; }
```

### Valore della proprietà

La stringa che rappresenta l'invio che rappresenta l'indirizzo email.

### Eccezioni

| eccezione | condizione |
| --- | --- |
| ArgumentException | genera se l'indirizzo e-mail non è in un formato riconosciuto. |

### Osservazioni

Quando si imposta un valore, vengono aggiornati anche i valori delle proprietà PR_SENT_REPRESENTING_SEARCH_KEY e PR_SENT_REPRESENTING_ENTRYID. Quando si imposta un valore nullo o una stringa vuota, i valori della proprietà vengono impostati su null.
