## PidTagPropertyDescriptor.Id property

Ottiene una quantità a 16 bit senza segno che identifica una proprietà con tag. Gli ID proprietà non sono necessariamente univoci. Ad eccezione degli ID proprietà nell'intervallo da 0x6800 a 0x7BFF, la combinazione di ID proprietà e tipo di dati è univoca. Gli ID proprietà nell'intervallo da 0x6800 a 0x7BFF sono definiti dalla classe di messaggio.
