## MailPrinter.CultureName property

Ottiene o imposta il nome della cultura nel formato languagecode2-country/regioncode2.

```csharp
public string CultureName { get; set; }
```

### Valore della proprietà

Una stringa che rappresenta le impostazioni cultura correnti.

### Osservazioni

languagecode2 è un codice di due lettere minuscolo derivato da ISO 639-1. country/regioncode2 deriva da ISO 3166 e di solito è composto da due lettere maiuscole.
