## ListFoldersReturnOptions enumeration

Opzioni di ritorno per l'operazione ListFolders Si prega di notare che queste opzioni sono supportate nel caso in cui il server supporti RFC 5258 "IMAP LIST Command Extensions" Vedere maggiori dettagli in https://tools.ietf.org/html/rfc5258

```csharp
[Flags]
public enum ListFoldersReturnOptions
```

### I valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| None | `0` | Non definito |
| Subscribed | `1` | SUBSCRIBED - fa in modo che il comando LIST restituisca lo stato di sottoscrizione per tutti i nomi delle cassette postali corrispondenti. L'attributo "\Subscribed" DEVE essere supportato e DEVE essere calcolato accuratamente quando viene specificata l'opzione di restituzione SUBSCRIBED. Inoltre, tutti i flag delle cassette postali DEVONO essere calcolati accuratamente (questo è diverso dal comportamento del comando LSUB). |
| Children | `2` | CHILDREN - richiede le informazioni sul figlio della cassetta postale. Questa opzione DEVE essere supportata da tutti i server. |
