## MapiPropertyFlags enumeration

Rappresenta i flag che possono essere impostati su una proprietà MAPI.

```csharp
[Flags]
public enum MapiPropertyFlags
```

### I valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| PROPATTR_MANDATORY | `1` | Se questo flag è impostato per una proprietà, quella proprietà NON DEVE essere cancellata dal file .msg (indipendentemente dalla memoria in cui è contenuta) e le implementazioni DEVONO restituire un errore se viene fatto qualsiasi tentativo in tal senso. Questo flag è impostato in circostanze in cui l'implementazione dipende dal fatto che la proprietà sia sempre presente nel file .msg una volta che è stato scritto lì. |
| PROPATTR_READABLE | `2` | Se questo flag non è impostato su una proprietà, tale proprietà non DEVE essere letta dal file .msg e le implementazioni DEVONO restituire un errore se si tenta di leggerlo. Questo flag è impostato su tutte le proprietà a meno che non vi sia un motivo specifico dell'implementazione per impedire la lettura di una proprietà dal file .msg. |
| PROPATTR_WRITABLE | `4` | Se questo flag non è impostato su una proprietà, tale proprietà non DEVE essere modificata o eliminata e le implementazioni DEVONO restituire un errore se viene fatto qualsiasi tentativo in tal senso. Questo flag è impostato in circostanze in cui l'implementazione dipende dal fatto che le proprietà siano scrivibili. |
