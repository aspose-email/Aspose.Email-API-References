## ProvisionPolicy.PolicyKey property

PolicyKey viene utilizzata dal server per contrassegnare lo stato delle impostazioni dei criteri sul client nella fase di download delle impostazioni del comando Provision. La PolicyKey è un elemento di tipo stringa con un massimo di 64 caratteri. Il valore della PolicyKey DOVREBBE essere una rappresentazione di stringa di un intero senza segno a 32 bit.
