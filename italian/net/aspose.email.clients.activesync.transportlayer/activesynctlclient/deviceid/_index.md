## ActiveSyncTLClient.DeviceID property

Un GUID che identifica il dispositivo. L'ID dispositivo è specificato dalla parte della regola ABNF specifica ID dispositivo del valore della query in testo normale. Il valore, rappresentato dalla regola ABNF device-id, è una stringa che specifica il dispositivo. Ogni dispositivo DEVE avere una stringa ID dispositivo univoca. Ogni richiesta dal dispositivo DEVE includere la stessa stringa ID dispositivo.
