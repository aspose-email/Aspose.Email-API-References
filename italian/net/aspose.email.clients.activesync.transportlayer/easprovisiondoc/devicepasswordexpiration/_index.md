## EASProvisionDoc.DevicePasswordExpiration property

Specifica il numero massimo di giorni prima della scadenza di una password. DevicePasswordExpiration può essere nullo, a indicare che non è stata impostata alcuna politica di scadenza della password. I valori validi sono elencati di seguito: = 0 - Le password non scadono. &gt; 0 - Le password scadono nel numero massimo specificato di giorni. Se DevicePasswordExpiration è nullo, un client DOVREBBE trattare questo valore come 0. Se il valore di DevicePasswordEnabled è impostato su FALSE, il client DOVREBBE ignorare questa proprietà.

public int? DevicePasswordExpiration { get; set; }
