## EASProvisionDoc.PasswordRecoveryEnabled property

Specifica se il server supporta la memorizzazione di una password di ripristino che deve essere inviata dal client utilizzando il comando Impostazioni. Una password di ripristino è una password speciale creata dal client che offre all'amministratore o all'utente la possibilità di accedere al dispositivo una volta , dopodiché all'utente viene richiesto di creare una nuova password. Il client crea quindi una nuova password di ripristino. Se PasswordRecoveryEnabled è impostato su TRUE, il server supporta la memorizzazione di una password di ripristino inviata dal dispositivo. Se la proprietà è impostata su FALSE, il dispositivo NON DEVE inviare una password di ripristino, perché il server non supporta la memorizzazione della password. Se PasswordRecoveryEnabled è nullo, un client DOVREBBE trattare questo valore come FALSO. Se il valore di DevicePasswordEnabled è FALSE, il client DOVREBBE ignorare questa proprietà. Questa proprietà DOVREBBE essere ignorata se il client non supporta le password di ripristino.
