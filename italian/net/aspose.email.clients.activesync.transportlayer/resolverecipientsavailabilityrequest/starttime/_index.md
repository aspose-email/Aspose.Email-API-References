## ResolveRecipientsAvailabilityRequest.StartTime property

Identifica l'inizio dell'intervallo di tempo libero/occupato richiesto dal client. StartTime non è supportato quando la versione del protocollo è 12.1. Se la disponibilità è inclusa nella richiesta, la richiesta DEVE includere anche StartTime e EndTime. Se il client invia un valore StartTime non valido, il server restituisce un valore di stato pari a 5 nella risposta al comando ResolveRecipients.

public DateTime StartTime { get; set; }
