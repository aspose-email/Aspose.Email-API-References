## PictureRequest class

Indica che il client sta richiedendo che le foto vengano restituite nella risposta del server. L'immagine non è supportata quando la versione del protocollo è 12.1 o 14.0. Contiene i dati relativi alla richiesta di foto.

## Proprietà

| Nome | Descrizione |
| --- | --- |
| [MaxPictures](../../aspose.email.clients.activesync.transportlayer/picturerequest/maxpictures) { get; set; } | Limita il numero di foto dei contatti restituite nella risposta del server. MaxPictures non è supportato quando la versione del protocollo è 12.1 o 14.0. Il server restituisce i primi N risultati con foto dei contatti, dove N è il valore di MaxPictures. Una volta raggiunto il limite di MaxPictures, il server restituisce il valore di stato 173 (NoPicture) se il contatto non ha foto, il valore di stato 174 (PictureTooLarge) se la foto del contatto è troppo grande o il valore di stato 175 (PictureLimitReached) se il contatto ha un foto ma è stato raggiunto il limite MaxPictures. Si noti che MaxPictures identifica il numero di foto dei contatti restituite per query. Pertanto, se il client include tre destinatari da risolvere e imposta il valore MaxPictures su 3 nella richiesta di comando ResolveRecipients, è possibile restituire un massimo di 9 foto di contatto. |
| [MaxSize](../../aspose.email.clients.activesync.transportlayer/picturerequest/maxsize) { get; set; } | Specifica la dimensione massima di una singola foto di contatto restituita nella risposta, in byte. MaxSize non è supportato quando la versione del protocollo è 12.1 o 14.0. Il valore massimo di MaxSize è 100 KB o 102400 byte. |
