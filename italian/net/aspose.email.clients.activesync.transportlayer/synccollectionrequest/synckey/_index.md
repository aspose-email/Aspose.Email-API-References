## SyncCollectionRequest.SyncKey property

Il valore SyncKey viene utilizzato dal server per contrassegnare lo stato di sincronizzazione di una raccolta. Una chiave di sincronizzazione di valore 0 (zero) inizializza lo stato di sincronizzazione sul server e provoca una sincronizzazione completa della raccolta.
