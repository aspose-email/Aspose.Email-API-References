## SmartRequest.Mime property

Contiene il messaggio con codifica MIME. Il contenuto Mime viene trasferito come BLOB opaco all'interno dei tag WBXML, come specificato in [WBXML1.2]. Se il messaggio contiene una convocazione di riunione, l'elemento Mime contiene i dettagli della riunione in Formato iCalendar [MS-OXCICAL] o formato TNEF (Transport Neutral Encapsulation Format) [MS-OXTNEF]. Come specificato in [RFC2447] sezione 3.4, le convocazioni di riunione di iCalendar hanno un tipo di contenuto di "testo/calendario" con il parametro del metodo impostato su "RICHIESTA".

public string Mime { get; set; }
