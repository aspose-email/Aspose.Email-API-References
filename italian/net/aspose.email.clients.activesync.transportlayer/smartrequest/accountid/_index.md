## SmartRequest.AccountId property

Identifica l'account da cui viene inviata un'e-mail. Se AccountId è nullo, il server invia l'e-mail utilizzando l'account identificato dalle impostazioni:PrimarySmtpAddress restituito nella risposta del comando Impostazioni. Se AccountId è incluso nella richiesta, il valore DEVE essere uguale a una delle impostazioni: valori dell'elemento AccountId inclusi nella risposta del comando Impostazioni. Il server DEVE convalidare che il valore dell'elemento AccountId fornito è valido per l'invio di e-mail. Viene restituito un valore di stato pari a 166 se il valore dell'elemento AccountId non è valido. Viene restituito un valore dell'elemento Status di 167 se l'account non supporta l'invio di e-mail. Nota Il server invia l'e-mail utilizzando l'account specificato da AccountId e non l'account specificato da From. L'elemento AccountId non è supportato quando la versione del protocollo è 12.1 o 14.0. Exchange 2007 restituisce il valore di stato 103 se l'elemento AccountId è incluso in una richiesta di comando SendMail, una richiesta di comando SmartForward o una richiesta di comando SmartReply.
