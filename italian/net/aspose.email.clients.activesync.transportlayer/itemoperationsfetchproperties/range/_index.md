## ItemOperationsFetchProperties.Range property

Specifica l'intervallo di byte che il client può ricevere in risposta all'operazione Recupero per un elemento della raccolta documenti. L'elemento Intervallo contiene un valore stringa nel formato mn, dove m minore di n e m è il valore minimo e n è il valore massimo. L'intervallo di byte è indicizzato a zero; il primo byte è indicato da uno 0 (zero). Se l'elemento Range viene omesso nella richiesta Fetch, viene recuperato l'intero elemento.

public string Range { get; set; }
