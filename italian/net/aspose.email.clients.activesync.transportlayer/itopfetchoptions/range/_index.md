## ItOpFetchOptions.Range property

Specifica l'intervallo di byte che il client può ricevere in risposta all'operazione di recupero per un elemento della raccolta documenti. L'elemento Range è un campo facoltativo. L'elemento Range contiene un valore stringa nel formato MN, dove M è minore di N e M è il valore minimo e N è il valore massimo. L'intervallo di byte è indicizzato a zero; il primo byte è indicato da uno 0 (zero). Se il Range è omesso nella richiesta Fetch, viene recuperato l'intero elemento. Se il FileReference è presente nella richiesta, allora l'elemento Range è l'unico elemento figlio valido di l'elemento Opzioni.

public string Range { get; set; }
