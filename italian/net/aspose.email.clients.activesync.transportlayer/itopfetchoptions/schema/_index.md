## ItOpFetchOptions.Schema property

Specifica lo schema dell'elemento da recuperare. Lo schema è supportato nelle opzioni per le richieste di recupero PIM (sezione 2.2.3.63.1). Non è supportato quando il client sta recuperando elementi da una raccolta documenti o un allegato. Se l'elemento Schema non è specificato, il server consente il recupero di tutte le proprietà.
