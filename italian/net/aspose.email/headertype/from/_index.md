## HeaderType.From property

Questo campo contiene l'identità della persona o delle persone che desideravano che questo messaggio fosse inviato. Il processo di creazione del messaggio dovrebbe per impostazione predefinita questo campo essere un unico indirizzo macchina autenticato, indicante l'AGENTE (persona, sistema o processo) che compone il messaggio. Se ciò non viene fatto, il campo "Mittente:" DEVE essere presente. Se il campo "Da:" è predefinito in questo modo, il campo "Mittente:" è facoltativo ed è ridondante con il campo "Da:". Esempio: Da: "Mr. Some One" qualcuno@somedomain.com
