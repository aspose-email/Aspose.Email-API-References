## ImapSpecialFolderTypes enumeration

Representerar en uppräkning av postlådor för specialanvändning Mer information finns i RFC6154 http://tools.ietf.org/html/rfc6154

public enum ImapSpecialFolderTypes

### Värderingar

| namn | Värde | Beskrivning |
| --- | --- | --- |
| NotSpecified | `0` | Om IMAP LIST-tillägget för brevlådor för specialanvändning (RFC6154) stöds betyder det att brevlådan är vanlig och inte används för speciella ändamål. |
| All | `1` | Denna brevlåda presenterar alla meddelanden i användarens meddelandelager. Implementationer KAN utelämna vissa meddelanden, som kanske de i \Trash och \Junk. När denna speciella användning stöds är det nästan säkert att det representerar en virtuell brevlåda. |
| Archive | `2` | Denna brevlåda används för att arkivera meddelanden. Innebörden av en "arkiverings"-postlåda är serverberoende; vanligtvis kommer den att användas för att få ut meddelanden från inkorgen, eller på annat sätt hålla dem ur användarens väg, samtidigt som de fortfarande görs tillgängliga. |
| Drafts | `3` | Den här brevlådan används för att hålla utkastmeddelanden, vanligtvis meddelanden som håller på att skapas men som ännu inte har skickats. I vissa serverimplementeringar kan detta vara en virtuell brevlåda som innehåller meddelanden från andra brevlådor som är markerade med "\Utkast" meddelandeflagga. Alternativt kan detta bara vara ett råd som en kund lägger ut utkast här. |
| Flagged | `4` | Denna brevlåda presenterar alla meddelanden som på något sätt är markerade som "viktiga". När denna speciella användning stöds representerar den sannolikt en virtuell brevlåda som samlar in meddelanden (från andra brevlådor) som är markerade med "\Flaggade" meddelandeflaggan. |
| Junk | `5` | Den här brevlådan är där meddelanden som anses vara skräppost förvaras. Vissa serverimplementationer kan placera meddelanden här automatiskt. Alternativt kan detta bara vara ett råd till ett skräppostfilter på klientsidan. |
| Sent | `6` | Den här brevlådan används för att hålla kopior av meddelanden som har skickats. Vissa serverimplementationer kan placera meddelanden här automatiskt. Alternativt kan detta bara vara ett råd att en klient sparar skickade meddelanden här. |
| Trash | `7` | Denna brevlåda används för att hålla meddelanden som har tagits bort eller markerats för radering. I vissa serverimplementationer kan detta vara en virtuell brevlåda, innehållande meddelanden från andra brevlådor som är markerade med meddelandeflaggan "\Deleted". Alternativt kan detta bara vara ett råd att en klient som väljer att inte använda IMAP "\Deleted"-modellen ska använda detta som sin papperskorgen. I serverimplementeringar som strikt förväntar sig IMAP "\Deleted"-modellen, kommer denna speciella användning sannolikt inte att stödjas. |
| Important | `8` | Denna brevlåda används för att hålla meddelanden som har markerats som viktiga. Postlådeattributet "\Important" är en signal om att brevlådan innehåller meddelanden som sannolikt är viktiga för användaren. I en implementering som också stöder "$Important" " nyckelord, representerar denna speciella användning sannolikt en virtuell brevlåda som samlar in meddelanden (från andra brevlådor) som är markerade med nyckelordet "$Important". I andra implementeringar kan systemet automatiskt lägga meddelanden där baserat på samma slags heuristik som noteras för nyckelordet "$Important". Skillnaderna mellan "\Important" och "\Flaggade" för postlådor liknar dem mellan "$Important" och "\Flaggade" för meddelanden. https://tools.ietf.org/html/rfc8457 |
