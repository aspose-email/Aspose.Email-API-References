## IGraphClient.CopyNotebook method

Kopierar en anteckningsbok till mappen Anteckningsböcker i målbiblioteket Dokument. Mappen skapas om den inte finns. För kopieringsoperationer följer du ett asynkront anropsmönster: Anrop först åtgärden Kopiera och efterfråga sedan operationens slutpunkt för resultatet. Permissions En av följande behörigheter krävs för att anropa detta API. Delegerat (arbets- eller skolkonto) Notes.Create, Notes.ReadWrite, Notes.ReadWrite.All Delegated(personligt Microsoft-konto) Notes.Create, Notes.ReadWrite Application Notes.ReadWrite.All_x000

```csharp
public string CopyNotebook(string itemId, string groupId, string renameAs)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| itemId | String | Artikel-id som ska kopieras |
| groupId | String | Id för gruppen som ska kopieras till. Använd endast när du kopierar till en Office 365-grupp. |
| renameAs | String | Namnet på kopian. Förinställer namnet på det befintliga objektet. |

### Returvärde

Om den lyckas returnerar den här metoden en Operation-Location-sträng. Du kan använda detta värde för att få status för operationen.
