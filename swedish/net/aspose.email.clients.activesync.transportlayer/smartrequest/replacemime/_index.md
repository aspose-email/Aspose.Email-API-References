## SmartRequest.ReplaceMime property

Anger om klienten skickar hela meddelandet. När ReplaceMime är TRUE, FÅR servern inte inkludera brödtexten eller bilagorna till det ursprungliga meddelandet som vidarebefordras. När ReplaceMime är FALSK, MÅSTE klienten lägga till brödtexten i det ursprungliga meddelandet som bilagor till det utgående meddelandet. Klienten kan använda den här egenskapen för att indikera om meddelandet redigerades inline, eller om meddelandet hade svars-/vidarebefordrantext framför källmeddelandet. Om ReplaceMime är TRUE, har meddelandet redigerats.

public bool ReplaceMime { get; set; }
