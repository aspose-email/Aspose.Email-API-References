## SyncRequest.HeartbeatInterval property

Anger antalet sekunder som servern SKA fördröja ett svar om inga nya objekt läggs till i de inkluderade mapparna. Giltiga värden för HeartbeatInterval är 60 till 3540 sekunder (59 minuter). När klienten begär ett intervall som ligger utanför det acceptabla intervallet kommer servern att skicka ett svar som inkluderar ett statusvärde på 14 och ett Limit-element. Antingen kan HeartbeatInterval eller Wait inkluderas i en synkkommandoförfrågan, men inte både. Om båda elementen ingår kommer serversvaret att innehålla ett statusvärde på 4.

public int? HeartbeatInterval { get; set; }
