## SyncCollectionRequest.DeletesAsMoves property

Förfrågningar som indikerar att alla borttagna objekt SKA flyttas till mappen Deleted Items. Om DeletesAsMoves-elementet är inställt på false är borttagningen permanent. Om klienten vill ta bort objekt permanent, MÅSTE begäran innehålla elementet DeletesAsMoves med värdet FALSE. Ett värde på TRUE, som är standard, indikerar att alla borttagna objekt flyttas till mappen Deleted Items.

public bool? DeletesAsMoves { get; set; }
