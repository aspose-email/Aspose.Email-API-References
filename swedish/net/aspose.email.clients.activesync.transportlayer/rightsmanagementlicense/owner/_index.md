## RightsManagementLicense.Owner property

Anger om användaren är ägaren till e-postmeddelandet. Värdet är TRUE om användaren är ägaren till e-postmeddelandet; annars FALSKT. Värdet TRUE indikerar att den autentiserade användaren har ägarrättigheter till detta meddelande. Detta element används endast för informationspresentation. De tillåtna elementen (EditAllowed, ReplyAllowed, etc.) används för att utvärdera om en viss åtgärd är tillåten eller begränsad.

public bool Owner { get; set; }
