## RightsManagementLicense.ExportAllowed property

Anger om IRM-skyddet på e-postmeddelandet kan tas bort av användaren. Värdet är TRUE om användaren kan ta bort IRM-skyddet när användaren vidarebefordrar, svarar eller svarar på allt på e-postmeddelandet; annars, FALSE. Om ett rättighetshanterat e-postmeddelande vidarebefordras eller besvaras med kommandot SmartForward eller SmartReply, utvärderas följande villkor: - Den ursprungliga rättighetspolicymallen har ExportAllowed-elementet satt till TRUE - TemplateID på den nya meddelande är satt till mallen "Ingen begränsning" (MallID-värde "00000000-0000-0000-0000-000000000000") Om båda villkoren är sanna tas IRM-skyddet bort från det utgående meddelandet. Det ursprungliga meddelandet behåller sitt IRM-skydd.
