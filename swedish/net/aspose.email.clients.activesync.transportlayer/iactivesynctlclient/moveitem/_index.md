## IActiveSyncTLClient.MoveItem method

Kommandot MoveItems flyttar ett eller flera objekt från en mapp på servern till en annan.

```csharp
public MoveItemResponse MoveItem(string srcMsgId, string srcFldId, string dstFldId)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| srcMsgId | String | Anger server-ID för objektet som flyttades. Elementvärdet SrcMsgId kan vara upp till 64 tecken långt. |
| srcFldId | String | Anger server-ID för källmappen (det vill säga mappen som innehåller objekten som ska flyttas). Elementvärdet SrcFldId kan vara upp till 64 tecken långt |
| dstFldId | String | Anger server-ID för målmappen (det vill säga mappen till vilken objekten flyttas). DstFldId-egenskapsvärdet kan vara upp till 64 tecken långt. |

Returnerar information som beskriver det flyttade objektet.
