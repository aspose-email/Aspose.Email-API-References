## DSNAction enumeration

Indikerar åtgärden som utförs av Reporting-MTA som ett resultat av dess försök att leverera meddelandet.

```csharp
public enum DSNAction
```

### Värderingar

| namn | Värde | Beskrivning |
| --- | --- | --- |
| None | `0` | Ingen av åtgärderna. |
| Delayed | `1` | Indikerar att rapporterings-MTA hittills inte har kunnat leverera eller vidarebefordra meddelandet, men det kommer att fortsätta att försöka göra det. Ytterligare aviseringsmeddelanden kan utfärdas när meddelandet försenas ytterligare eller levereras, eller om leveransförsök senare avbryts. |
| Delivered | `2` | Indikerar att meddelandet har levererats till den mottagaradress som angetts av avsändaren, vilket inkluderar "leverans" till en e-postlistexploderare. Det betyder inte att meddelandet har lästs. Detta är ett terminaltillstånd och ingen ytterligare DSN för denna mottagare bör förväntas. |
| Expanded | `3` | Indikerar att meddelandet har levererats till mottagaradressen som specificerats av avsändaren och vidarebefordrats av Reporting-MTA bortom den destinationen till flera ytterligare mottagaradresser. Ett åtgärdsvärde för "expanderat" skiljer sig från Levererat genom att "expanderat" inte är ett terminaltillstånd. Ytterligare "misslyckade" och/eller "försenade" meddelanden kan tillhandahållas. |
| Failed | `4` | Indikerar att meddelandet inte kunde levereras till mottagaren. Rapporterings-MTA har övergett alla försök att leverera meddelandet till denna mottagare. Inga ytterligare meddelanden bör förväntas. |
| Relayed | `5` | Indikerar att meddelandet har vidarebefordrats eller gatewayerats till en miljö som inte tar ansvar för att generera DSN vid framgångsrik leverans. Detta åtgärdsvärde SKA INTE användas om inte avsändaren har begärt meddelande om lyckad leverans för denna mottagare. |
