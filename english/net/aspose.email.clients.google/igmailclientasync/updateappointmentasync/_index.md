## IGmailClientAsync.UpdateAppointmentAsync method

Asynchronously updates an appointment in the specified calendar.

```csharp
public Task<Appointment> UpdateAppointmentAsync(string calendarId, Appointment appointment, 
    CancellationToken cancellationToken = default)
```

| Parameter | Type | Description |
| --- | --- | --- |
| calendarId | String | The ID of the calendar. |
| appointment | Appointment | The appointment to update. |
| cancellationToken | CancellationToken | A cancellation token. |

### Return Value

A task that represents the asynchronous operation. The task result contains the updated [`Appointment`](../../../aspose.email.calendar/appointment/).
