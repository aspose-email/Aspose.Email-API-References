## IActivityClient.StopSubscription method

Dieser Vorgang beendet ein Abonnement für den angegebenen Inhaltstyp. Wenn ein Abonnement beendet wird, erhalten Sie keine Benachrichtigungen mehr und können keine verfügbaren Inhalte abrufen. Wenn das Abonnement später neu gestartet wird, haben Sie Zugriff auf neue Inhalte von diesem Zeitpunkt an. Sie können keine Inhalte abrufen, die zwischen dem Zeitpunkt der Beendigung und dem Neustart des Abonnements verfügbar waren.

```csharp
public void StopSubscription(string contentType)
```

| --- | --- | --- |
| contentType | String | Gibt den Inhaltstyp an. |
