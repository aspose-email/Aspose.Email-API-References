## ExchangeAdminActivity.ExternalAccess property

Gibt an, ob das Cmdlet von einem Benutzer in Ihrer Organisation, von Microsoft-Rechenzentrumsmitarbeitern oder einem Rechenzentrumsdienstkonto oder von einem delegierten Administrator ausgeführt wurde. Der Wert False gibt an, dass das Cmdlet von jemandem in Ihrer Organisation ausgeführt wurde. Der Wert True gibt an, dass das Cmdlet von Rechenzentrumsmitarbeitern, einem Rechenzentrumsdienstkonto oder einem delegierten Administrator ausgeführt wurde.

public bool ExternalAccess { get; set; }
