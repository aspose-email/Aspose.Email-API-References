## MapiRecipient.EmailAddress property

Ruft die E-Mail-Adresse des Empfängers oder Absenders der -Nachricht ab oder legt sie fest.

```csharp
public string EmailAddress { get; set; }
```

### Eigentumswert

Die E-Mail-Adresse.

### Ausnahmen

| Ausnahme | Bedingung |
| --- | --- |
| ArgumentNullException | löst aus, wenn die Empfängeradresse null oder leer ist. |
| ArgumentException | löst aus, wenn die Empfängeradresse nicht in einem erkannten Format vorliegt. |

### Bemerkungen

Beachten Sie, dass diese Eigenschaft eine Adresse im SMTP-Format anzeigt. Das bedeutet, wenn MapiRecipient.AddressType == "EX" und MapiPropertyTag.PR_EMAIL_ADDRESS eine Adresse im Exchange-Stil enthalten, wird der Wert für die Eigenschaft mapiRecipient.EmailAddress aus PR_SMTP_ADDRESS. ausgewählt. Wenn ein Wert festgelegt wird, werden die Werte von PR_SEARCH_KEY Die Eigenschaften und PR_ENTRYID werden ebenfalls aktualisiert.
