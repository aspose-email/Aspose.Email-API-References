## MapiPropertyTag.PR_ATTACH_CONTENT_ID field

Enthält den Content Identification Header eines MIME-Nachrichtenanhangs. Diese Eigenschaft wird für die MHTML-Unterstützung verwendet.

```csharp
public static readonly long PR_ATTACH_CONTENT_ID;
```

### Bemerkungen

Es kann verwendet werden, um zwischen Inline-Anhängen wie Signaturbildern und normalen angehängten Dateien zu unterscheiden. Wenn Anhang diese Eigenschaft hat, dann ist es ein eingebettetes Bild.
