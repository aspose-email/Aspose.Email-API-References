## MapiMessageItemBase.Subject property

Ruft den Betreff der Nachricht ab oder legt ihn fest.

```csharp
public string Subject { get; set; }
```

### Eigentumswert

Die Zeichenfolge, die den Betreff der Nachricht darstellt.

### Bemerkungen

Beim Festlegen eines Werts werden die Werte der Eigenschaften SubjectPrefix(PR_SUBJECT_PREFIX) und NormalizedSubject(PR_NORMALIZED_SUBJECT) ebenfalls aktualisiert. Wenn Betreff kein Präfix hat, wird der Wert der Eigenschaft SubjectPrefix auf null gesetzt. Beim Festlegen eines Nullwerts oder einer leeren Zeichenfolge werden die Werte der Eigenschaften Subject, SubjectPrefix und NormalizedSubject auf null gesetzt.
