## MapiMessageItemBase.BodyRtf property

Ruft den RTF-formatierten Nachrichtentext ab oder legt ihn fest.

```csharp
public string BodyRtf { get; set; }
```

### Eigentumswert

Die Zeichenfolge, die den Nachrichtentext darstellt rtf.

### Bemerkungen

Beim Festlegen eines Werts werden die Werte der Eigenschaften PR_RTF_COMPRESSED, PR_RTF_DECOMPRESSES und PR_BODY aktualisiert. Ein festgelegter Zeichenfolgenwert muss im RTF-Format vorliegen. Wenn also ein Wert im HTML-Format festgelegt werden muss, muss der Wert zuerst in RTF codiert werden, entsprechend der RTF-Erweiterungsspezifikation. Um den Inhalt der Textnachricht schnell im HTML- oder Nur-Text-Format festzulegen, verwenden Sie bitte die SetBodyContent-Methode. Beim Festlegen eines Nullwerts oder einer leeren Zeichenfolge werden die Werte der BodyRtf- und Body-Eigenschaften auf null gesetzt.
