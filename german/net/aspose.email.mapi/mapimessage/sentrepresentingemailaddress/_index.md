## MapiMessage.SentRepresentingEmailAddress property

Ruft die E-Mail-Adresse für den Messaging-Benutzer ab, der durch den Absender dargestellt wird, oder legt diese fest.

```csharp
public string SentRepresentingEmailAddress { get; set; }
```

### Eigentumswert

Die Zeichenfolge, die die gesendete E-Mail-Adresse repräsentiert.

### Ausnahmen

| Ausnahme | Bedingung |
| --- | --- |
| ArgumentException | löst aus, wenn die E-Mail-Adresse kein anerkanntes Format hat. |

### Bemerkungen

Beim Festlegen eines Werts werden die Werte der Eigenschaften PR_SENT_REPRESENTING_SEARCH_KEY und PR_SENT_REPRESENTING_ENTRYID ebenfalls aktualisiert. Beim Festlegen eines Nullwerts oder einer leeren Zeichenfolge werden die Werte der Eigenschaft auf Null gesetzt.
