## MapiMessage.RemoveAttachments method

Entfernt alle Anhänge aus den angegebenen Outlook-Nachrichtendateien.


| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| path | String | Der Name der Outlook-Nachrichtendatei. |

### Rückgabewert

### Beispiele

Das folgende Beispiel zeigt, wie Anhänge in Outlook-Nachrichtendateien zerstört werden.

[C#]

[Visual Basic]

```csharp
//Anhänge aus Outlook-Nachrichtendateien entfernen
MapiAttachmentCollection attachments = MapiMessage.RemoveAttachments(@"c:\outlookmessage.msg");

//Anhänge
foreach(MapiAttachment att in attachments)
{
   Console.WriteLine("Attachment Name:"+att.FileName);
   att.Save(att.FileName);
}
```
