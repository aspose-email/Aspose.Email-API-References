## FileCompatibilityMode enumeration

Definiert innere Konvertierungen, die notwendigerweise beim Laden von oder Speichern einer Nachricht durchgeführt werden müssen. Standardmäßig wird CR nicht durch CRLF ersetzt, Tnef-Anhang wird nicht gespeichert.

```csharp
[Flags]
public enum FileCompatibilityMode
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| None | `0` | Es sind keine Werte gesetzt. |
| AllowCROnly | `1` | Zeigt die Notwendigkeit des zwangsweisen Ersetzens von Zeilentrennzeichen durch CRLF. |
| PreserveTnefAttachments | `2` | Steuert das Lade-/Speicherverhalten von TNEF-Anhängen |
