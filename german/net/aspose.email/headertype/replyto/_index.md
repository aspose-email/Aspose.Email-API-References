## HeaderType.ReplyTo property

Dieses Header-Feld soll angeben, wohin der Absender Antworten senden möchte. Leider ist dies nicht eindeutig, da es verschiedene Arten von Antworten gibt, die der Absender möglicherweise an verschiedene Adressen senden möchte. Insbesondere gibt es persönliche Antworten, die nur für eine Person bestimmt sind, und Gruppenantworten, die für die gesamte Gruppe von Personen bestimmt sind, die die beantwortete Nachricht lesen (häufig kann eine Mailingliste, ein Newsgroup-Name wegen unterschiedlicher Syntax hier nicht erscheinen, siehe " FollowupTo" .).

public static HeaderType ReplyTo { get; }
