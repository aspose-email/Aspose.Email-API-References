## HeaderType.DispositionNotificationTo property

Wenn das DispositionNotificationTo-Feld gesetzt ist, wird eine Anfrage für eine MDN (Message Delivery Notification) gestellt. Die E-Mail-Software des Empfängers (Outlook, Eudora usw.) kann die Anfrage stillschweigend ignorieren oder den Benutzer um Erlaubnis bitten, die MDN zu senden. Es gibt keine Garantie für den "Rückerhalt". Das DispositionNotificationTo-Feld ist der De-facto-Standard für die Anforderung von Empfangsbestätigungen (dh MDN oder Benachrichtigungen über die Zustellung von Nachrichten).

public static HeaderType DispositionNotificationTo { get; }
