## SmtpClient.SucceededSending event

Fügt Abonnenten für Benachrichtigungen über das erfolgreiche Senden von E-Mail-Nachrichten hinzu oder entfernt sie. Bitte beachten Sie, dass Benachrichtigungen von anderen Threads aufgerufen werden, daher sollten Sie eine Thread-sichere Ereignisbehandlungsroutine implementieren. Beachten Sie auch, dass dieses Ereignis für die E-Mail-Client-Instanz gleich ist, Falls der Client im Mehrfachverbindungsmodus läuft, werden Nachrichten von allen Verbindungen durch dieses Ereignis verarbeitet. Der Kunde muss also in seinem Code Nachrichten von Ereignissen analysieren, um zu verstehen, welche Nachricht sich auf welche Operation bezieht.

```csharp
public event EventHandler<MailMessageEventArgs> SucceededSending;
```
