## DSNAction enumeration

Gibt die Aktion an, die vom Reporting-MTA als Ergebnis seines Versuchs, die Nachricht zuzustellen, ausgeführt wird.

```csharp
public enum DSNAction
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| None | `0` | Keine der Aktionen. |
| Delayed | `1` | Gibt an, dass der meldende MTA die Nachricht bisher nicht zustellen oder weiterleiten konnte, dies aber weiterhin versuchen wird. Zusätzliche Benachrichtigungen können ausgegeben werden, wenn die Nachricht weiter verzögert oder erfolgreich zugestellt wird oder wenn Zustellversuche später abgebrochen werden. |
| Delivered | `2` | Gibt an, dass die Nachricht erfolgreich an die vom Absender angegebene Empfängeradresse zugestellt wurde, einschließlich "Zustellung" an einen Mailinglisten-Explorer. Es zeigt nicht an, dass die Nachricht gelesen wurde. Dies ist ein Endzustand und es sollte kein weiterer DSN für diesen Empfänger erwartet werden. |
| Expanded | `3` | Gibt an, dass die Nachricht erfolgreich an die vom Absender angegebene Empfängeradresse zugestellt und vom Reporting-MTA über dieses Ziel hinaus an mehrere zusätzliche Empfängeradressen weitergeleitet wurde. Ein Aktionswert von „erweitert“ unterscheidet sich von geliefert darin, dass „erweitert“ kein Endzustand ist. Weitere "fehlgeschlagene" und/oder "verzögerte" Benachrichtigungen können bereitgestellt werden. |
| Failed | `4` | Zeigt an, dass die Nachricht nicht an den Empfänger zugestellt werden konnte. Der meldende MTA hat alle Versuche abgebrochen, die Nachricht an diesen Empfänger zuzustellen. Es sind keine weiteren Benachrichtigungen zu erwarten. |
| Relayed | `5` | Gibt an, dass die Nachricht weitergeleitet oder in eine Umgebung geleitet wurde, die keine Verantwortung für die Generierung von DSNs bei erfolgreicher Zustellung übernimmt. Dieser Aktionswert SOLLTE NICHT verwendet werden, es sei denn, der Absender hat eine Benachrichtigung über die erfolgreiche Zustellung für diesen Empfänger angefordert. |
