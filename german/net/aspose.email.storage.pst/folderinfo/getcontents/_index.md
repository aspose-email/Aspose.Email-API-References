```csharp
public MessageInfoCollection GetContents(int startIndex, int count)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| startIndex | Int32 | Der Startnachrichtenindex. |
| count | Int32 | Die Anzahl der Nachrichten, die abgerufen werden. |

### Rückgabewert

Sammlung von MessageInfo.

### Ausnahmen

| Ausnahme | Bedingung |
| --- | --- |
| ArgumentOutOfRangeException | löst aus, wenn startIndex kleiner als 0 ist. |
| ArgumentOutOfRangeException | löst aus, wenn startIndex größer oder gleich der Gesamtzahl der Nachrichten ist, die dieser Ordner enthält. |

### Bemerkungen

Wenn der Parameter "count" kleiner als 0 oder größer als die Anzahl der verbleibenden Nachrichten ist, wird die Anzahl der verbleibenden Nachrichten zurückgegeben.
