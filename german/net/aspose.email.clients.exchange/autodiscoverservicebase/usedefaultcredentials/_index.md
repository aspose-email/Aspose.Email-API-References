## AutodiscoverServiceBase.UseDefaultCredentials property

Ruft einen Wert ab oder legt einen Wert fest, der angibt, ob die Anmeldeinformationen des derzeit bei Windows angemeldeten Benutzers zur -Authentifizierung bei den Exchange-Webdiensten verwendet werden sollen. Wenn UseDefaultCredentials auf true gesetzt wird, wird die Eigenschaft Credentials automatisch auf null gesetzt.

public bool UseDefaultCredentials { get; set; }
