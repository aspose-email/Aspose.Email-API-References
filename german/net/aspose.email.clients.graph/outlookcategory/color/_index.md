## OutlookCategory.Color property

Zeichenkettendarstellung einer voreingestellten Farbkonstante, die eine Kategorie charakterisiert und die einer von 25 vordefinierten Farben zugeordnet ist. Die möglichen Werte für Farbe sind voreingestellte Konstanten wie Keine, Voreinstellung0 und Voreinstellung1. Jede voreingestellte Konstante wird weiter einer Farbe zugeordnet; die tatsächliche Farbe hängt vom Outlook-Client ab, in dem die Kategorien angezeigt werden.

public string Color { get; set; }
