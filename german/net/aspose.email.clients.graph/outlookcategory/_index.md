## OutlookCategory class

Stellt eine Kategorie dar, nach der ein Benutzer Outlook-Elemente wie Nachrichten und Ereignisse gruppieren kann. Der Benutzer definiert Kategorien in einer Hauptliste und kann eine oder mehrere dieser benutzerdefinierten Kategorien auf ein Element anwenden. https://docs.microsoft.com/en-us/graph/api/resources/outlookcategory?view =graph-rest-1.0
