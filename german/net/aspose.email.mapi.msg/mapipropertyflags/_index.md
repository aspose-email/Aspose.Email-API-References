## MapiPropertyFlags enumeration

Stellt Flags dar, die für eine MAPI-Eigenschaft festgelegt werden können.

```csharp
[Flags]
public enum MapiPropertyFlags
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| PROPATTR_MANDATORY | `1` | Wenn dieses Flag für eine Eigenschaft gesetzt ist, DARF diese Eigenschaft NICHT aus der .msg-Datei gelöscht werden (unabhängig davon, in welchem Speicher sie enthalten ist), und Implementierungen MÜSSEN einen Fehler zurückgeben, wenn dies versucht wird. Dieses Flag wird in Fällen gesetzt, in denen die Implementierung davon abhängt, dass diese Eigenschaft immer in der .msg-Datei vorhanden ist, sobald sie dort geschrieben wurde. |
| PROPATTR_READABLE | `2` | Wenn dieses Flag für eine Eigenschaft nicht gesetzt ist, DARF diese Eigenschaft nicht aus der .msg-Datei gelesen werden, und Implementierungen MÜSSEN einen Fehler zurückgeben, wenn versucht wird, sie zu lesen. Dieses Flag wird für alle Eigenschaften gesetzt, es sei denn, es gibt einen implementierungsspezifischen Grund, der verhindert, dass eine Eigenschaft aus der .msg-Datei gelesen wird. |
| PROPATTR_WRITABLE | `4` | Wenn dieses Flag für eine Eigenschaft nicht gesetzt ist, DARF diese Eigenschaft nicht geändert oder gelöscht werden, und Implementierungen MÜSSEN einen Fehler zurückgeben, wenn dies versucht wird. Dieses Flag wird gesetzt, wenn die Implementierung davon abhängt, dass die Eigenschaften beschreibbar sind. |
