| Ok | `1` | Die OK-Antwort weist auf eine Informationsnachricht vom Server hin. Wenn markiert, zeigt es den erfolgreichen Abschluss des zugehörigen Befehls an. Die Form ohne Tag weist auf einen Fehler auf Protokollebene hin, für den der zugehörige Befehl nicht bestimmt werden kann; es kann auch auf einen internen Serverfehler hinweisen.
