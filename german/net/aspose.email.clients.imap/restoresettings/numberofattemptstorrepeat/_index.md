## RestoreSettings.NumberOfAttemptsToRrepeat property

Erhält oder setzt einen Wert, der die Anzahl der Versuche definiert, einen fehlgeschlagenen Vorgang zu wiederholen Falls ein IMAP-Befehl innerhalb eines Sicherungsvorgangs ein fehlgeschlagenes Ergebnis zurückgibt, versucht der IMAP-Client, diesen Vorgang entsprechend der definierten Anzahl von Malen erneut zu wiederholen. Wenn beispielsweise der FETCH-Vorgang einen Fehler zurückgibt AE_1_1_0243 FETCH 219 (BODY) AE_1_1_0243 NO[UNAVAILABLE] FETCH Service ist vorübergehend nicht verfügbar Der Client versucht erneut, ihn auszuführen.

public int NumberOfAttemptsToRrepeat { get; set; }
