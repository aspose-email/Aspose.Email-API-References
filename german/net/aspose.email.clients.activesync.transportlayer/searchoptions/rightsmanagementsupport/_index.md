## SearchOptions.RightsManagementSupport property

Gibt an, wie der Server rechteverwaltete E-Mail-Nachrichten an den Client zurückgibt. Wenn der Wert TRUE ist, dekomprimiert und entschlüsselt der Server rechteverwaltete E-Mail-Nachrichten, bevor er sie an den Client sendet. Wenn der Wert FALSE ist, dekomprimiert oder entschlüsselt der Server rechteverwaltete E-Mail-Nachrichten nicht, bevor er sie an den Client sendet. Wenn das RightsManagementSupport-Element nicht in einer Anforderungsnachricht enthalten ist, wird ein Standardwert von FALSE angenommen.
