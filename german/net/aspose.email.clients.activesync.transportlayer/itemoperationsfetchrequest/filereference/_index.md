## ItemOperationsFetchRequest.FileReference property

Gibt einen eindeutigen Bezeichner an, der vom Server jedem Anhang zugewiesen wird. In einer ItemOperations-Anforderung kann nur ein FileReference-Bezeichner pro Abruf vorhanden sein. Ein Verstoß gegen diese Einschränkung führt dazu, dass der Statuselementwert 2 vom Server zurückgegeben wird. Der Client kann jedoch mehrere Anhänge abrufen, indem er einen Fetch-Knoten pro Anhang verwendet.

public string FileReference { get; set; }
