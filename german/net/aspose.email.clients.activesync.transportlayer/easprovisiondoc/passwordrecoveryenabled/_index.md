## EASProvisionDoc.PasswordRecoveryEnabled property

Gibt an, ob der Server die Speicherung eines Wiederherstellungskennworts unterstützt, das vom Client mit dem Befehl „Einstellungen“ gesendet wird. Ein Wiederherstellungskennwort ist ein spezielles Kennwort, das vom Client erstellt wird und dem Administrator oder Benutzer die Möglichkeit gibt, sich einmalig am Gerät anzumelden , , wonach der Benutzer aufgefordert wird, ein neues Passwort zu erstellen. Der Client erstellt dann ein neues Wiederherstellungskennwort. Wenn PasswordRecoveryEnabled auf TRUE gesetzt ist, unterstützt der Server die Speicherung eines vom Gerät gesendeten Wiederherstellungskennworts. Wenn die Eigenschaft auf FALSE gesetzt ist, SOLLTE das Gerät KEIN Wiederherstellungskennwort senden, da der Server die Speicherung des Kennworts nicht unterstützt. Wenn PasswordRecoveryEnabled null ist, SOLLTE ein Client diesen Wert als FALSE behandeln. Wenn der Wert von DevicePasswordEnabled FALSE ist, SOLLTE der Client diese Eigenschaft ignorieren. Diese Eigenschaft SOLLTE ignoriert werden, wenn der Client keine Wiederherstellungskennwörter unterstützt.
