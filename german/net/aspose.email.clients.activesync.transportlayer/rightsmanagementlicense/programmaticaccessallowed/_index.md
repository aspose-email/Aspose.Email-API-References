## RightsManagementLicense.ProgrammaticAccessAllowed property

Gibt an, ob Anwendungen von Drittanbietern programmgesteuert auf den Inhalt der E-Mail-Nachricht zugreifen können. Der Wert ist TRUE, wenn Anwendungen von Drittanbietern programmgesteuert auf den Inhalt der E-Mail-Nachricht zugreifen können; andernfalls FALSCH. Der Wert TRUE gibt an, ob der geschützte Inhalt für andere Anwendungen zugänglich ist. Geschützter Inhalt besteht aus dem Nachrichtentext und Anhängen.

public bool ProgrammaticAccessAllowed { get; set; }
