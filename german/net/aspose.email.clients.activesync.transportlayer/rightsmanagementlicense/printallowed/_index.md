## RightsManagementLicense.PrintAllowed property

Gibt an, ob die E-Mail vom Benutzer gedruckt werden kann. Dieses Element gibt keine Clientunterstützung zum Drucken einer E-Mail-Nachricht an; es gibt nur an, ob die E-Mail-Nachricht gedruckt werden kann, wenn der Client das Drucken unterstützt. Der Wert ist TRUE, wenn die E-Mail vom Benutzer gedruckt werden kann; andernfalls FALSE.

public bool PrintAllowed { get; set; }
