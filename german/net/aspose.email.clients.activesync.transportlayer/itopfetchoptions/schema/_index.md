## ItOpFetchOptions.Schema property

Gibt das Schema des abzurufenden Elements an. Das Schema wird in den Optionen für PIM-Abrufanforderungen (Abschnitt 2.2.3.63.1) unterstützt. Es wird nicht unterstützt, wenn der Client Elemente aus einer Dokumentbibliothek oder einen Anhang abruft. Wenn das Schema-Element nicht angegeben ist, lässt der Server zu, dass alle Eigenschaften abgerufen werden.
