## ItOpFetchOptions.Range property

Gibt den Bytebereich an, den der Client als Antwort auf den Fetch-Vorgang für ein Dokumentbibliothekselement empfangen kann. Das Bereichselement ist ein optionales Feld. Das Bereichselement enthält einen Zeichenfolgenwert im Format MN, wobei M kleiner als N ist und M der Mindestwert und N der Höchstwert ist. Der Bytebereich ist nullindiziert; das erste Byte wird durch eine 0 (Null) angezeigt. Wenn der Range in der Fetch-Anforderung weggelassen wird, wird das gesamte Element abgerufen. Wenn die FileReference in der Anforderung vorhanden ist, dann ist das Range-Element das einzige gültige untergeordnete Element von das Options-Element.

public string Range { get; set; }
