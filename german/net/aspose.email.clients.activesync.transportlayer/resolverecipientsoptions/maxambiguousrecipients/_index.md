## ResolveRecipientsOptions.MaxAmbiguousRecipients property

Begrenzt die Anzahl der Vorschläge, die für jeden mehrdeutigen Empfängerknoten in der Antwort zurückgegeben werden. Der Wert von MaxAmbiguousRecipients ist auf einen Bereich von 0–9999 begrenzt. Jeder mehrdeutige Empfängerknoten erhält nur so viele Vorschläge und nicht mehr. Die in RecipientCount zurückgegebene Empfängeranzahl kann vom Client verwendet werden, um die Gesamtzahl der für diesen Empfänger verfügbaren Vorschläge zu ermitteln.

```csharp
public int? MaxAmbiguousRecipients { get; set; }
```
