## ResolveRecipientsOptions.MaxCertificates property

Begrenzt die Gesamtzahl der Zertifikate, die vom Server zurückgegeben werden. Der Wert von MaxCertificates ist auf einen Bereich von 0–9999 begrenzt. Dieses Limit stellt sicher, dass kein einzelner Empfänger einen unvollständigen Satz von Zertifikaten erhält. Wenn das MaxCertificates-Limit erreicht wird, während Zertifikate für eine Adressliste aufgelistet werden, erhält diese Adressliste keine Zertifikate zurück und ein Statuswert von 8 wird zurückgegeben.

public int? MaxCertificates { get; set; }
