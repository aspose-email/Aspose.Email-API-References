## SyncCollectionRequest.SyncKey property

Der SyncKey-Wert wird vom Server verwendet, um den Synchronisierungsstatus einer Sammlung zu markieren. Ein Synchronisierungsschlüssel mit dem Wert 0 (Null) initialisiert den Synchronisierungsstatus auf dem Server und bewirkt eine vollständige Synchronisierung der Sammlung.
