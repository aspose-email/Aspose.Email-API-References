## ProvisionPolicy.PolicyKey property

PolicyKey wird vom Server verwendet, um den Status der Richtlinieneinstellungen auf dem Client in der Downloadphase der Einstellungen des Bereitstellungsbefehls zu markieren. Der PolicyKey ist ein Element vom Typ String mit maximal 64 Zeichen. Der Wert des PolicyKey SOLLTE eine Stringdarstellung einer 32-Bit-Ganzzahl ohne Vorzeichen sein.
