## SyncOptions.RightsManagementSupport property

Gibt an, wie der Server rechteverwaltete E-Mail-Nachrichten an den Client zurückgibt. Wenn der Wert dieses Elements TRUE ist, dekomprimiert und entschlüsselt der Server rechteverwaltete E-Mail-Nachrichten, bevor er sie an den Client sendet, wie in Abschnitt 3.2.4.3 angegeben . Wenn der Wert FALSE (0) ist, dekomprimiert oder entschlüsselt der Server rechteverwaltete E-Mail-Nachrichten nicht, bevor er sie an den Client sendet. Wenn das RightsManagementSupport-Element nicht definiert ist, wird ein Standardwert von FALSE angenommen.

public bool? RightsManagementSupport { get; set; }
