## IActiveSyncTLClient.MoveItem method

Der MoveItems-Befehl verschiebt ein oder mehrere Elemente von einem Ordner auf dem Server in einen anderen.

```csharp
public MoveItemResponse MoveItem(string srcMsgId, string srcFldId, string dstFldId)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| srcMsgId | String | Gibt die Server-ID des verschobenen Elements an. Der SrcMsgId-Elementwert kann bis zu 64 Zeichen lang sein. |
| srcFldId | String | Gibt die Server-ID des Quellordners an (d. h. des Ordners, der die zu verschiebenden Elemente enthält). Der Wert des SrcFldId-Elements kann bis zu 64 Zeichen lang sein |
| dstFldId | String | Gibt die Server-ID des Zielordners an (d. h. des Ordners, in den die Elemente verschoben werden). Der DstFldId-Eigenschaftswert kann bis zu 64 Zeichen lang sein. |

### Rückgabewert

Gibt Informationen zurück, die das verschobene Element beschreiben.
