## IActiveSyncTLClient.HeartbeatInterval property

Das HeartbeatInterval-Element ist ein untergeordnetes Element des Ping-Elements in Ping-Befehlsanfragen und -antworten. In Ping-Befehlsanforderungen gibt es die Zeitspanne in Sekunden an, die der Server warten soll, bevor er eine Antwort sendet, wenn keine neuen Elemente zu den angegebenen Ordnern hinzugefügt werden, wie in Abschnitt 3.1.5.6 angegeben. Das HeartbeatInterval-Element wird ebenfalls vom Server mit einem Statuscode von 5 zurückgegeben und gibt entweder das minimale oder maximale Intervall an, das zulässig ist, wenn der Client ein Heartbeat-Intervall angefordert hat, das außerhalb des akzeptablen Bereichs liegt.

public int HeartbeatInterval { get; set; }
