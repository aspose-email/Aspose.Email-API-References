Der maximale Wert für WindowSize ist 512. Wenn WindowSize jedoch auf 512 festgelegt ist, kann der Server Sync-Antwortnachrichten senden die weniger als 512 Aktualisierungen enthalten. Wenn der Server nicht alle Updates in einer einzigen Nachricht sendet, enthält die Sync-Antwortnachricht MoreAvailable, was darauf hinweist, dass auf dem Server zusätzliche Updates zum Herunterladen auf den Client vorhanden sind.
