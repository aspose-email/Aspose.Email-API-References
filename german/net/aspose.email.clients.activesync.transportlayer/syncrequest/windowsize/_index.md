## SyncRequest.WindowSize property

Gibt eine maximale Anzahl geänderter Elemente in einer Sammlung oder einer Anforderung an, die in die Synchronisierungsantwort aufgenommen werden SOLLTEN. Der maximale Wert für WindowSize ist 512. Wenn WindowSize jedoch auf 512 festgelegt ist, kann der Server Sync-Antwortnachrichten senden die weniger als 512 Aktualisierungen enthalten. Wenn der Server nicht alle Updates in einer einzigen Nachricht sendet, enthält die Sync-Antwortnachricht MoreAvailable, was darauf hinweist, dass auf dem Server zusätzliche Updates zum Herunterladen auf den Client vorhanden sind. Die WindowSize erscheint nur in gesendeten Anforderungen vom Client zum Server. Wird die WindowSize weggelassen, verhält sich der Server so, als ob ein WindowSize-Element mit dem Wert 100 übergeben würde. Der Server interpretiert den Wert 0 (null) und Werte über 512 als 512.
