## SearchResult.LongId property

Gibt eine eindeutige Kennung an, die vom Server jedem zurückgegebenen Ergebnissatz zugewiesen wird. Der Wert der LongId kann als lange ID verwendet werden, die in der ItemOperations-Befehlsanforderung, der SmartReply-Befehlsanforderung, der SmartForward-Befehlsanforderung oder angegeben ist die MeetingResponse-Befehlsanforderung, auf die Ergebnismenge zu verweisen. Der Client MUSS den Wert von LongId als undurchsichtige Zeichenfolge mit bis zu 256 Zeichen speichern.

public string LongId { get; set; }
