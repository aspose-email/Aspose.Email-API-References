## ItOpMoveOptions.MoveAlways property

Gibt an, ob die angegebene Konversation, einschließlich aller zukünftigen E-Mails in der Konversation, immer in den durch die DstFldId angegebenen Ordner verschoben werden soll. MoveAlways MUSS in einer ItemOperations-Anforderung festgelegt werden, wenn eine Verschiebungsoperation für eine Konversation durchgeführt wird. Ein Statuselementwert von 155 wird zurückgegeben, wenn das MoveAlways-Element in der ItemOperations-Anforderung für einen Verschiebevorgang nicht festgelegt ist.

public bool? MoveAlways { get; set; }
